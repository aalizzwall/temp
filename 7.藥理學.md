# 📚 CHAPTER 1: 藥理學的一般原理 (General Pharmacology) 💊🔬
# ============================================

### **藥物動力學 vs 藥物動力學：**

#### **藥物動力學 (Pharmacokinetics) - 「身體對藥物的處理」：**
1. **吸收 (Absorption)**：口服→胃排空速度、腸道血流、胃酸度影響
2. **分佈 (Distribution)**：蛋白結合率、脂溶性影響穿越血腦屏障
3. **代謝 (Metabolism)**：肝臟 CYP450 系統（誘導/抑制）
4. **排泄 (Excretion)**：腎小球過濾 + 腎小管分泌

#### **[Image Prompt: Pharmacokinetics ADME diagram showing Absorption (GI tract with absorption curves), Distribution (blood-brain barrier penetration by lipophilic drugs), Metabolism (liver CYP450 enzymes), and Excretion (kidney glomerular filtration), labeled educational flowchart]**

### **藥物動力學參數記憶：**
- **Bioavailability (F)**：口服吸收程度（首過效應影響）
- **Half-life (t½)**：消除一半所需時間（4-5 個 t½達到穩定濃度）
- **Clearance (CL)**：清除率（肝/腎功能指標）

#### **穩定狀態血中濃度公式：**
```
Css = (F × Dose) / (CL × τ)
其中：
F = 生物利用率
Dose = 劑量
CL = 清除率
τ = 給藥間隔
```

### **題 2：抗憂鬱藥啟動期管理**
> 32 歲女性新診斷重度憂鬱症，使用 SSRIs（如 Sertraline）。治療第 3 天患者表示失眠加重、焦慮增加、輕度頭痛。這是什麼現象？如何處理？

#### ✅ **答案分析：**
- **啟動期副作用 (Activation Syndrome)**：前 1-2 週常見
- **處理策略**：
  1. 告知患者這是正常適應過程
  2. 可暫時增加劑量緩慢滴定

  4. 避免酒精

### **臨床案例題 4：高血壓患者用藥調整**
> 62 歲男性，初診收縮壓 160 mmHg、舒張壓 95 mmHg。無糖尿病或腎病變。您會選擇哪一類藥物作為起始治療？

#### ✅ **答案分析：**
- **首選**：ACEI（如 Lisinopril）或 ARB（如 Valsartan）
- **理由**：初診高血壓 + 無併發症 → 可單用 ACEI/ARB
- **備選**：CCB（如 Amlodipine）→ 若患者不耐受乾咳

### **臨床案例題 5：糖尿病腎病變患者降壓治療**
> 55 歲女性，Type 2 糖尿病 + CKD（eGFR 35 mL/min），目前使用 Metformin、Lisinopril。血壓收縮壓 145 mmHg、舒張壓 88 mmHg。如何調整？

#### ✅ **答案分析：**
1. **ACEI (Lisinopril)**：✅ 繼續使用（腎臟保護）
2. **添加ARB？** ⚠️ 避免雙重 ACEI/ARB（增加腎毒性風險）
3. **首選方案**：Lisinopril + CCB（如 Amlodipine）→ 增強降壓效果
