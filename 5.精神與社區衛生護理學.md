## 🌟 一、精神疾病護理 (High Yield)

### 1.1 精神分裂症 (Schizophrenia)

#### 🌟 核心考點

**【診斷要點】**
- **Flesch Reading Ease**: 7.2 (中等難度)
- **Key Symptoms**:
  - 阳性症状：幻聽、幻覺、妄想、思维混亂
  - 阴性症状：情感平淡、社交退縮、意志缺乏
- **病程**: 至少持續 6 個月才算慢性化

**【藥物治療】💊**
```
第一代抗精神病藥 (典型)：
├── Haldol (Haloperidol) — 首選急性期躁動
├── Thorazine (Chlorpromazine) — 嗜睡副作用大
└── Trilafon (Fluphenazine) — 半衰期長

第二代抗精神病藥 (非典型)：
├── Risperdal (Risperidone) — 首選，代謝副作用小
├── Seroquel (Quetiapine) — 嗜睡、低血壓風險
├── Zyprexa (Olanzapine) — 體重增加、糖尿病風險高
└── Geodon (Ziprasidone) — QTc 延長風險，需監測

💡 高頻考點：
- Risperdal 是唯一代謝副作用較小的非典型藥物
- Clozapine 用於治療抵抗型精神分裂症（必須定期測白血球）
```

**【副作用記憶法則】** 🧠
```
「5D+1H」：
├── Dry (乾) — 流體分泌減少
├── Dizzy (暈) — 低血壓
├── Dysphagia (吞嚥困難) — 自主神經障礙
├── Depressed Mood (情緒低落) — 內臟性幻聽後的情緒問題
├── Diabetes (糖尿病) — 體重增加/代謝綜合症
└── Hyperprolactinemia (高泌乳激素) — 溢乳/陽萎

📝 臨床警示：老年人使用第一代藥物需特別注意「Dizziness + Dry」，容易跌倒！
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **自殺風險評估**
   - 🧠 **重點記憶**：精神分裂症病人**病情穩定後反而較易自殺**（急性期以被害妄想為主）
   - ⏰ **觀察指標**：治療中藥效過渡期最危險

2. **藥源性肺炎風險**
   - 🩺 長期服用抗精神病藥者，需定期檢查肺部功能
   - 💊 Olanzapine、Clozapine 較高風險

3. **代謝綜合症監測**
   ```
   Zyprexa + Seroquel → 高風險群體
   └── 需追蹤：體重/BMI/血糖/血脂/HOMA-IR
   ```

4. **藥物相互作用**
   - 🔄 CYP450 系統代謝影響：
     - Olanzapine (CYP1A2, 2D6) → 與 SSRIs 合併使用需小心
     - Quetiapine (3A4) → 與 Statins 有相互作用

#### ❌ 易錯點

- **❌ 錯誤觀念**：「精神分裂症病人需要隔離治療」
  - ✅ **正確認識**：社會功能恢復才是治療目標，應鼓勵社區融合

- **❌ 錯誤觀念**：「所有抗精神病藥都會導致體重增加」
  - ✅ **正確認識**：Risperidone、Ziprasidone 的風險較低

### 1.2 雙相情緒障礙 (Bipolar Disorder)

#### 🌟 核心考點

**【疾病分期】**
```
躁狂期 → 混合型發作 → 輕度躁狂/輕躁 → 抑鬱發作 → 快速循環型
     ↑      ↑           ↑        ↑            ↑
┌─────┼─────┬───────────┴────────┼─────────┐
│ 躁狂期：情緒高昂、活動增加、睡眠需求減少
├─────┤                            │        │
│混合型：躁狂與抑鬱症狀同時存在
└─────┘                            │        │
                                   │        │
【重要記憶點】：雙相 disorder 最常併發的是「快速循環型」，需長期維持治療
```

**【藥物選擇策略】** 💊
```
心境穩定劑：
├── Lithium (碳酸鋰) — 首選經典藥，預防自殺風險最高
│   └── 監測指標：Serum Level (0.6-1.2 mEq/L), TSH, Creatinine
├── Valproate (丙戊酸) — 女性育齡期避免，致畸性高
├── Carbamazepine (卡馬西平) — 肝毒性需監測
└── Lamotrigine (拉莫三嗪) — 抗抑鬱效果好，但啟動要慢（每週加減量）

📝 臨床警示：Lithium 中毒警訊 = 震顫 + 吞嚥困難 + 意識模糊
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **藥物毒副作用監測**
   - 🧪 Lithium：腎功能/甲狀腺/血藥濃度 (0.6-1.2 mEq/L)
   - 💊 Valproate：肝功能/血小板數量

2. **自殺風險評估**
   - 🧠 **重點記憶**：雙相病人**從躁狂期轉換到抑鬱期時**是最高風險階段！

3. **電療適應症**
   - ⚡ 適用於：治療抵抗型、快速循環型、嚴重發作期

#### ❌ 易錯點

- **❌ 錯誤觀念**：「抗焦慮藥物可以長期用於雙相情緒障礙」
  - ✅ **正確認識**：苯二氮芪類只用於急性期，需搭配心境穩定劑長期治療

### 1.3 憂鬱症 (Major Depressive Disorder)

#### 🌟 核心考點

**【診斷要點】**
- **DSM-5 Criteria**: 至少 5 項症狀，包括：
  - 情緒低落或興趣丧失（2項必須）
  - 睡眠改變
  - 食慾改變
  - 疲勞/精力不足
  - 自我貶低/罪惡感
  - 思考能力下降
  - 自傷/自殺意念

**【藥物選擇策略】** 💊
```
SSRIs (首選)：
├── Sertraline (Zoloft) — 首選，適合合併焦慮
├── Escitalopram (Lexapro) — 副作用最小，老年人首選
└── Citalopram (Celexa) — QTc 延長風險

SNRIs:
├── Venlafaxine (Effexor) — 高劑量才有 SNRI 效果
└── Duloxetine (Cymbalta) — 適合合併慢性疼痛

Bupropion (Wellbutrin):
└── 不影響性功能，適合合併吸煙癮斷戒需求

📝 臨床警示：抗抑鬱藥物需「啟動期 (2-4週)」才見效，避免病人中途中斷！
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **自殺風險評估**
   - 🧠 **重點記憶**：治療初期（第 2-4 週）情緒改善前，自殺意念可能反而增強！
   - 📋 **監測指標**：每週一次自殺風險評估表 (SISSI)

2. **藥物停用症候群**
   ```
   Paroxetine + Sertraline → 高風險（需慢慢減量）
   Fluoxetine + Bupropion → 較低風險（長半衰期）
   ```

3. **血清素症候群警訊** 🚨
   ```
   Triad Symptoms:
   ├── 自主神經興奮：心悸、發汗、震顫
   ├── 精神狀態改變：焦慮、意識混亂
   └── 肌肉超活化：肌攣、反射亢進
   
   💡 處理：停藥 + 支援性護理，嚴重者需 ICU 觀察
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「抗抑鬱藥物沒有成癮問題」
  - ✅ **正確認識**：雖無苯二氮芪類的依賴性，但停藥仍需慢慢減量

- **❌ 錯誤觀念**：「所有憂鬱症都需要抗鬱劑治療」
  - ✅ **正確認識」：輕度憂鬱可用心理治療 + 生活型態調整

## 🌟 二、社區衛生護理 (High Yield)

### 2.1 傳染病社區照護 🦠

#### 🌟 核心考點

**【高優先級傳染病】**
```
HIV/AIDS:
├── PEP (暴露後預防): 72 小時內開始，服用 4 週
├── PrEP (暴露前預防): 每月檢查 HIV RNA + 腎功能
└── ART (抗逆轉錄病毒治療): 三種藥物組合

📝 臨床警示：HIV+ 孕婦可透過 C-section 降低垂直傳染率至 <1%!

Tuberculosis (TB):
├── Initial Phase: RIPE (Rifampin, Isoniazid, Pyrazinamide, Ethambutol)
├── Continuation Phase: RH (Rifampin, Isoniazid)
└── 監測指標：肝功能 + 視神經（Ethambutol）

📝 臨床警示：異物性結膜炎需隔離治療，避免傳染他人！
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **抗逆轉錄病毒藥物副作用**
   ```
   Integrase Inhibitors (首選):
   ├── Dolutegravir — 最常見，副作用小
   └── Raltegravir — 懷孕安全

   Protease Inhibitors:
   └── 高膽固醇/胰島素抵抗風險

   💡 監測：每 3 個月一次完整血液檢查 + 腎功能
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「HIV 陽性者無法正常工作」
  - ✅ **正確認識**：現代 ART 治療下，病毒載量檢測不到=不傳染他人（U=U）

---

### 2.2 慢性病管理 🏙️

#### 🌟 核心考點

**【糖尿病社區照護】**
```
Medication Hierarchy:
├── Metformin (首選) — GI副作用 + B12缺乏風險
├── SGLT2 Inhibitors (新選擇) — 腎病/心臟病保護
├── GLP-1 Agonists — 體重減少效果好
└── Insulin — 終末期需使用

💡 臨床警示：SGLT2 抑制劑有酮酸中毒風險（即使血糖正常）!

【高血壓社區照護】
├── First Line: ACEI / ARB / CCB / Thiazide
├── Second Line: Combinaion Therapy (雙重治療)
└── Goal BP: <130/80 mmHg (高齡者可放宽至<140/90)

📝 臨床警示：ACEI + ARB 組合不可同時使用（腎衰竭風險）!
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **藥效相互作用監測**
   - 🔄 Metformin + IV造影劑 → 急性肾衰竭風險
   - 💊 Insulin + 感染/壓力 → 需調整劑量

2. **藥物依從性評估**
   ```
   高流失率群體：
   ├── 年輕族群（認為自己不會生病）
   └── 多重共病者（服用>5 種藥物）
   
   💡 解決方案：簡化製程 + 藥學教育
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「血糖正常就不用吃降糖藥」
  - ✅ **正確認識**：藥物是為了保護器官功能，不能隨意停藥

## 🌟 三、精神疾病社區復健 🔄

### 3.1 復健原則

#### 🌟 核心考點

**【復健階段劃分】**
```
急性期 → 過渡期 → 社區融合期 → 自我管理期
    ↑       ↑        ↓            ↓
┌─────┼─────┬────────┴──────────┐
│ 醫院治療 +                          │ 社會支持系統建立
│ 家屬心理教育                         │
└─────┘                            │
                                   │
【重點記憶】：社區復健的**核心**是「賦權（Empowerment）」!
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **藥物減量策略**
   ```
   Step-down Strategy:
   ├── 保持原藥劑量的 50% → 2-3 週
   ├── 轉換為長效製程 → 1-2 週
   └── 維持最低有效劑量 → 長期追蹤
   
   💡 臨床警示：突然停藥可能引發「撤藥症候群」!
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「精神疾病病人沒有工作能力」
  - ✅ **正確認識**：適度復健可恢復社會功能，需早期介入

## 📝 考前速記卡 🃏

### 🧠 重點記憶清單

| 疾病/藥物 | 首選藥物 | 主要副作用 | 監測指標 |
|-----------|----------|------------|----------|
| 精神分裂症 | Risperidone | 代謝綜合症 | BMI/Lipid/Glucose |
| 雙相情緒障礙 | Lithium | 腎毒性/甲減 | Serum Level/TSH |
| 憂鬱症 | Escitalopram | 血清素症候群 | SISSI 評估表 |
| HIV/AIDS | Integrase Inhibitors | 頭痛/腹瀉 | Viral Load/RNA |
| 糖尿病 | Metformin | GI 副作用/B12 缺乏 | HbA1c/B12 Level |

### 🚨 高風險情境速記卡

- **精神疾病病人** → 治療中藥效過渡期最易自殺
- **HIV+ 孕婦** → C-section 可降低垂直傳染率至 <1%
- **SGLT2 抑制劑** → 即使血糖正常也有酮酸中毒風險
- **抗抑鬱藥物** → 啟動期（2-4 週）自杀意念反而可能增強
- **ACEI + ARB** → 不可同時使用（腎衰竭風險）

```
├── Dry (乾) — 流體分泌減少
├── Dizzy (暈) — 低血壓

├── Depressed Mood (情緒低落) — 內臟性幻聽後的情緒問題
├── Diabetes (糖尿病) — 體重增加/代謝綜合症

```
