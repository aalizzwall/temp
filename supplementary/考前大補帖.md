## 💊 考科一：基礎醫學 (搶救失憶專區)

### 1. 絕對必考解毒劑 (秒殺題)
| 中毒/過量藥物 | 唯一指定解毒劑 |
| :--- | :--- |
| **普拿疼 (Acetaminophen)** | **Acetylcysteine (NAC)** |
| **嗎啡 / 鴉片類 (Morphine)** | **Naloxone (Narcan)** |
| **安眠藥 (BZD類, -pam)** | **Flumazenil** |
| **毛地黃 (Digoxin)** | Digoxin immune Fab (Digibind) |
| **抗凝血劑：肝素 (Heparin)** | **魚精蛋白 (Protamine sulfate)** |
| **抗凝血劑：華法林 (Warfarin)** | **維生素 K (Vitamin K)** |
| **有機磷 / 農藥** | **Atropine** + PAM |

### 2. 生理與微免黃金考點
* **四大過敏反應：**
  * Type I (立即型)：IgE。(過敏性休克、氣喘)。
  * Type II (細胞毒性)：IgG/IgM。(輸血錯誤、溶血)。
  * Type III (免疫複合體)：(SLE 紅斑性狼瘡、風濕性關節炎)。
  * Type IV (遲發型)：**T細胞 (唯一無關抗體)**。(PPD 結核測試、接觸性皮膚炎)。
* **血壓急救 RAAS 機制順序：** 腎素(腎) ➔ 血管收縮素原(肝) ➔ ACE(肺) ➔ 血管收縮素II (最強收縮) ➔ 醛固酮(留鈉排鉀)。

## 🛏️ 考科二：基本護理學與護理行政 (常規與法則)

### 1. 防護裝備 (PPE) 穿脫順序
* **穿 (由下而上/內而外)：** 髮帽 ➔ 口罩 ➔ 隔離衣 ➔ 護目鏡 ➔ 手套。
* **脫 (最髒的先脫)：** **手套 (絕對第一)** ➔ 護目鏡 ➔ 隔離衣 ➔ 口罩 ➔ 髮帽。

### 2. 壓傷 (Pressure Injury) 分期關鍵字
* **第一期：** 皮膚完整，指壓**不會變白**的紅斑。
* **第二期：** 表淺破皮或**水泡** (無腐肉)。
* **第三期：** 看到**皮下脂肪**。
* **第四期：** 看到**骨頭、肌肉、肌腱**。
* **深部組織損傷 (DTI)：** 皮膚完整，但呈現**紫色/黯紅色**。

### 3. 品管與時數
* **事後算帳：** RCA (根本原因分析)，找系統錯，畫魚骨圖。
* **事前防範：** HFMEA (失效模式效應分析)。
* **護理時數 (NHPPD) 計算：** `(白+小夜+大夜總人數 * 8小時) ÷ 病人總數`。

## 🫀 考科三：內外科護理學 (生命交關特區)

### 1. 致命數值與體徵對比
| 項目 | 關鍵特徵與處置 |
| :--- | :--- |
| **IICP (腦壓升高)** | 庫欣氏三徵：**血壓↑、心跳↓**、呼吸不規則。床頭抬高 30 度。 |
| **休克 (Shock)** | **血壓↓、心跳↑**。平躺、下肢抬高 (除心因性外)。 |
| **左心衰竭** | 塞在肺：端坐呼吸、**粉紅色泡沫痰**。 |
| **右心衰竭** | 塞在全身：頸靜脈怒張、周邊水腫、肝脾大。 |

### 2. 糖尿病急症 (首要處置皆為：**大量給予 0.9% NaCl 生理食鹽水**！)
* **DKA (Type 1)：** 庫斯毛耳氏呼吸 (深快)、呼氣水果味、有酮體。
* **HHS (Type 2)：** 極度高血糖 (>600)、嚴重脫水、**無酮體**。

### 3. ABG 判讀口訣 (ROME)
* **R**espiratory **O**pposite：pH 與 PaCO2 方向相反 ➔ 呼吸性。
* **M**etabolic **E**qual：pH 與 HCO3- 方向相同 ➔ 代謝性。

## 👶 考科四：產兒科護理學 (關鍵時機點)

### 1. 胎心音減速 (VEAL CHOP) 🌟🌟🌟 (每年必考)
* **V**ariable (變異性) ➔ **C**ord (臍帶受壓) ➔ 處置：**膝胸臥式** / 改變姿勢。
* **E**arly (早期) ➔ **H**ead (胎頭受壓) ➔ 處置：正常生理現象，繼續觀察。
* **A**cceleration (加速) ➔ **O**K (正常) ➔ 處置：不需處理。
* **L**ate (晚期) ➔ **P**lacenta (胎盤灌流不足) ➔ 處置：**最危險！立即左側臥、給氧、停催產素**。

### 2. 兒科必考指標
* **川崎氏症：** 大劑量 IVIG + **高劑量 Aspirin** (唯一兒童發燒可給阿斯匹靈的特例)。
* **法洛氏四重症 (TOF)：** 缺氧發作時，病童會**蹲踞 (Squatting)**；護理給予**膝胸臥式**。
* **生長指標：** 體重 5 個月變 2 倍；1 歲變 3 倍。前囟門 12-18 個月閉合 (凹陷=脫水，膨出=IICP)。

## 🌳 考科五：精神與社區衛生護理 (層次與分類)

### 1. 抗精神病藥物副作用 (EPS) 發生時間序
1. **急性肌張力不全：** 最快發生(幾天)，眼球上吊、斜頸。最危急。
2. **靜坐不能：** 幾週發生，不斷來回走動。
3. **假性巴金森氏症：** 面具臉、小碎步。
4. **遲發性運動不能 (TD)：** 最晚發生(數年)，不自主噘嘴吐舌。不可逆。

### 2. 社區三段五級預防 (判斷題秒殺)
* **第一級 (健康促進)：** 衛教、營養。
* **第二級 (特殊保護)：** **打疫苗**、戴安全帽。
* **第三級 (早期診斷/治療)：** **所有癌症篩檢**、新生兒篩檢、抹片檢查。
* **第四級 (限制殘障)：** 預防併發症 (如中風後防攣縮)。
* **第五級 (復健)：** 職能治療、庇護工場。
