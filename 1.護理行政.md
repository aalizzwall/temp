## 🌟 一、護理品質管理 (High Yield)

### 1.1 品質指標監測 🔍

#### 🌟 核心考點

**【高優先級指標】**
```
結構指標 (Structure):
├── 設備完善度：ICU 床位比 / 護理師/醫師比
├── 人員配置：每病房護理師數 ≥ 4.0:1
└── 環境安全：跌倒風險評估表完整率

過程指標 (Process):
├── 手衛生執行率：≥90% (WHO 標準)
├── 壓瘡預防：翻身紀錄完整率 100%
├── 靜脈炎預防：沖洗時長 ≥15min/3h
└── 感染控制：隔離措施執行率 100%

結果指標 (Outcome):
├── 再發病率：<5% (內科病房)
├── 死亡率：目標<8.67%(台灣平均基準線)
└── 滿意度：≥90%(護理長評鑑標準)
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **跌倒風險評估 (Morse Fall Scale)**
   ```
   高風險群體 (>50 分):
   ├── 老年病人 (>75 歲) — +24 分
   ├── 過去有跌倒史 — +13 分
   ├── 使用安樂椅/輪椅 — +6 分
   └── 服用藥物 (類固醇/抗高血壓藥) — +5 分
   
   💡 處理：紅標警示 + 床緣鈴 + 防滑襪
   ```

2. **壓瘡風險評估 (Braden Scale)**
   ```
   高危群體 (<12 分):
   ├── 感知覺受損 — 1-3 分 (最高)
   ├── 活動力受限 — 1-3 分
   ├── 移動力受限 — 1-3 分
   ├── 營養狀態差 — 1-2 分
   ├── 濕度控制不佳 — 1-3 分
   └── 摩擦剪力和剪切力 — 1-3 分
   
   💡 處理：每 2 小時翻身 + 氣墊床 + 局部護理
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「只有長期臥床的病人會壓瘡」
  - ✅ **正確認識**：高齡/營養不良/藥物副作用都可能增加風險

### 1.2 醫療安全事件預防 🛡️

#### 🌟 核心考點

**【高發事件類型】**
```
TOP5 醫療安全事故:
├── 给药錯誤 (Medication Error): 30-40%
├── 跌倒/撞傷 (Fall Injury): 15-20%
├── 靜脈炎/感染 (Phlebitis/Infection): 10-15%
├── 壓瘡 (Pressure Ulcer): 8-10%
└── 手術部位錯誤 (Wrong Site Surgery): <1%(JCI 標準)

📝 JCI 標準：醫療不良事件通報率必須≥95%
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **靜脈治療相關風險**
   ```
   非血管內治療 (NVCD) 優先順序:
   ├── 外周靜脈導管 (PVC) — 首選短期治療
   ├── 中心靜脈導管 (CVC) — 高風險/長期治療
   └── 動脈導管 (Arterial Line) — 僅限血壓監測
   
   💡 臨床警示：CVC 置入需嚴格無菌技術 + 定期換藥！
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「所有病人都需要中心靜脈導管」
  - ✅ **正確認識**：僅高風險藥物/長期治療才需要，外周導管優先

## 🌟 二、護理人力配置 (High Yield)

### 2.1 排班與人力管理 👥

#### 🌟 核心考點

**【人力配置標準】**
```
台灣衛福部標準:
├── ICU: 1.0-1.5 床/護理師
├── 一般病房: 4.0-4.5 床/護理師
└── 急診: 需配合時段人力配置

💡 高風險時段:
├── 白班結束前 2 小時 (最易疏漏)
├── 交班時 (資訊交接不完整)
└── 夜班 (注意力下降 + 疲勞累積)
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **夜班安全事故率**
   ```
   夜班高風險事件:
   ├── 跌倒/撞傷：較白班增加 40%
   ├── 给药錯誤：較白班增加 35%
   └── 感染控制疏失：較白班增加 30%
   
   💡 處理策略：
   ├── 減少連續夜班次數 (≤2 次/週)
   ├── 夜班前充分休息 (≥6 小時)
   └── 交班時使用「SBAR」標準溝通模式
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「資深護理師可以一個人處理所有工作」
  - ✅ **正確認識**：合理配置 + 團隊合作才能確保病人安全

---

### 2.2 護理長職責與權限 👔

#### 🌟 核心考點

**【護理長法定職責】** (台灣法規)
```
護理法 (Health Care Providers Act):
├── 排班規劃與人力調配
├── 醫療不良事件處理流程
├── 護理品質監測指標分析
└── 護理人員在職培訓規劃

💡 重點記憶：護理長需定期向院方回報「病人滿意度」+「不良事件統計」!
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **護理品質異常時**
   ```
   護理長必做事項:
   ├── 立即啟動改善計畫 (QCC)
   ├── 通報醫療品質委員會
   └── 召開部門會議討論對策
   
   📝 臨床警示：連續 3 天不良事件率超過門檻 → 必須進行根本原因分析 (RCA)!
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「護理長只需要管理人力」
  - ✅ **正確認識**：品質管理 + 教育培訓 + 風險評估是核心職責

## 🌟 三、護理法規與倫理 ⚖️

### 3.1 護理法與病人權益 📜

#### 🌟 核心考點

**【台灣護理法重點章節】**
```
主要法規:
├── 醫師法：醫師執業規範 + 處方權限
├── 藥師法：藥品管理 + 調劑流程
├── 醫療法：醫療機構運作標準
├── 身心障礙保護法：特殊需求病人照護
└── 感染控制相關法規 (傳染病防治法)

💡 高頻考點：護理人員「拒絕執行」不合法/不合倫理處方之權利!
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **護病比不足時**
   ```
   護理長權限:
   ├── 可要求調派其他單位人力支援
   ├── 可啟動「緊急醫療資源調度」
   └── 若無法改善 → 需書面記錄並回報院方
   
   📝 臨床警示：護病比嚴重失衡時 → 護理師有「停止接收新病人」之權利!
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「護理師必須執行醫師所有處方」
  - ✅ **正確認識**：不合法/不合倫理處方 → 可拒絕 + 回報醫療團隊討論

---

### 3.2 病人自主權與知情同意 📋

#### 🌟 核心考點

**【知情同意五要件】**
```
1. 資訊完整：清楚說明治療目的、風險、替代方案
2. 理解程度：確認病人真正理解（非只是簽名）
3. 自願意願：無脅迫/誘導因素
4. 能力評估：病人有理解決策之能力
5. 同意表達：明確表示接受治療

💡 重點記憶：護理師需記錄「已向病人確認理解」!
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **無行為能力病人**
   ```
   替代決策人優先順序:
   ├── 法定監護人 (第一順位)
   ├── 近親屬/配偶
   └── 法院指定之人
   
   💡 臨床警示：若無人可當 → 需向法院聲請指定臨時監護人!
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「病人簽名就等於同意」
  - ✅ **正確認識**：需確認理解程度 + 自願意願 (不能只是形式化流程)

## 📝 考前速記卡 🃏

### 🧠 重點記憶清單

| 類別 | 標準/指標 | 備註 |
|------|----------|------|
| 手衛生執行率 | ≥90% (WHO 標準) | 院內品質指標 |
| ICU 護病比 | ≤1.5:1 | 台灣衛福部標準 |
| 一般病房護病比 | ≤4.5:1 | 夜間需加強配置 |
| 壓瘡翻身頻率 | 每 2 小時 | 高風險病人需更頻繁 |
| 靜脈炎沖洗時長 | ≥15min/3h | 標準作業流程 |
| 手冊記錄完整率 | ≥95% | JCI 認證標準 |

### 🚨 高風險情境速記卡

- **跌倒風險 (>50 分 Morse)** → 紅標警示 + 床緣鈴 + 防滑襪
- **壓瘡風險 (<12 分 Braden)** → 每 2 小時翻身 + 氣墊床
- **夜班高風險事件** → 跌倒/撞傷 (+40%) / 给药錯誤 (+35%)
- **連續 3 天不良事件超標** → 必須進行根本原因分析 (RCA)

### 💡 臨床警示速記卡

```
【靜脈治療優先順序】:
PVC (外周) > CVC (中心) > Arterial Line (動脈)

【護理長權限】:
1. 調派人力支援
2. 啟動緊急醫療資源調度
3. 書面記錄並回報院方

【拒絕執行處方】:
不合法/不合倫理 → 可拒絕 + 回報團隊討論
```
