基本護理學 📚

🌟 核心考點總覽 (涵蓋 110-115 年國考)

本主題重點：以下知識點在護理師國家考試中頻繁出現，需特別記憶與理解！

A 環境準備要求
  • 手消毒、穿戴無菌手套、滅菌器械準備
B 清創處理順序
  • 由內至外（傷口中心→緣部）、先乾淨後髒污
C 廢棄物分類處置
  • 醫療廢棄物袋標記、焚化處理流程規範
A 風險評估工具 (Braden 量表)
  • 4 級以上需介入：翻身、氣墊床、營養補充
B 翻身頻率標準
  • 2 小時一次（嚴重風險）、30°側身角度
C 壓瘡分級判定
  • I 期：局部紅斑不褪色、II 期：水疱形成、III 期：皮膚壞死
A 靜脈穿刺途徑選擇
  • 肘正中靜脈（粗、直）、前臂靜脈（適合長期輸液）
B 輸液速率控制
  • 微調整流速夾、輸液泵設定精度監控
C 靜脈炎預防措施
  • 生理食鹽水稀釋刺激性藥劑、定期更換針頭（每 72 小時）
A 體溫測量標準部位
  • 腋溫＜口溫 0.5℃、肛溫最準確（口腔感染不宜）
B 發燒處理原則
  • ＞38.5℃給予退燒藥（acetaminophen）、物理降溫法
C 體溫監測頻率
  • 急性期每 4-6 小時、穩定後可減少至每日 2-3 次
A 氣管插管護理重點
  • 氣囊壓力維持（25-30cmH₂O）、吸痰無菌操作
B 呼吸器設定標準
  • 潮氣量 6-8ml/kg、PEEP 維持肺泡張力
C 肺部感染預防
  • 翻身拍背（每 2 小時）、痰液引流技術應用
A 清創處理標準程序
  • 移除異物、切除壞死組織、使用抗菌敷料（銀離子）
B 縫合技術選擇
  • 皮層縫合（深層對合）、皮下縫合（減壓張力）
C 感染預防措施
  • 維持清潔乾燥、定期換藥、觀察紅腫熱痛徵象
A 每日熱量需求計算
  • BMR×活動係數、特殊族群（孕婦/乳母）調整公式
B 蛋白質攝取標準
  • 0.8-1.2g/kg/day、腎病病患需限制攝取量
C 特殊飲食適應
  • 低鹽飲食（＜2g/day）、低脂飲食（心疾病患）
A 尿管護理重點
  • 集尿袋低於膀胱水平、每日沖洗保持管腔暢通
B 尿失禁處理原則
  • 吸收性墊材、皮膚保護霜（氧化鋅）塗抹
C 泌尿道感染預防
  • 多飲水（2000-3000ml/day）、不憋尿習慣養成
A 採血操作規範
  • 洗手消毒、採血部位（指尖→耳垂）、採血量控制
B 正常血糖範圍
  • 空腹：70-100mg/dl、餐後：＜120mg/dl
C 低血糖處理流程
A 七對確認標準
  • 姓名、藥物、劑量、途徑、時間、濃度、過敏史
B 注射針頭選擇
  • IM：23-25G（肌肉深層）、SQ：27-30G（皮下組織）
C 藥物相容性檢查
  • 混合前確認不相容、避光保存需避光注射器

🔍 考題回顧 (110-115 年精選)
【110 年】基本護理學
(題目：壓瘡預防應注意事項？
(A)每 12 小時翻身一次 (B)每 2 小時翻身並保護骨突處 (C)使用普通枕頭墊高患側 (D)保持傷口潮濕)
正解：(B) 每 2 小時翻身並保護骨突處
解析：壓瘡預防重點為定期翻身（2 小時）、使用氣墊床、減壓裝置應用。⚠️ 常考重點：Braden 量表評估與介入標準。
【111 年】基本護理學
(題目：靜脈留置針應多久更換一次？
(A)72 小時 (B)96 小時 (C)120 小時 (D)依醫師指示延長)
正解：(A) 72 小時
解析：一般靜脈留置針建議每 72-96 小時更換，感染高風險者需更頻繁評估。⚠️ 常考重點：靜脈通路維護與感染預防措施。
【112 年】基本護理學
(題目：氣管切開病人吸痰無菌操作時，吸痰管插入深度約為？
(A)5-8cm (B)10-14cm (C)15-20cm (D)超過 20cm)
正解：(B) 10-14cm
解析：吸痰深度應為氣管套管至聲門間距離，過深傷及支氣管、過淺無法有效清除分泌物。⚠️ 常考重點：呼吸照護無菌操作標準。
【113 年】基本護理學
(題目：病人體溫 39.5℃且脈搏細速，最可能處於何種狀態？
(A)代償期休克 (B)熱衰竭前兆 (C)高血壓危象 (D)甲狀腺機能低下)
正解：(B) 熱衰竭前兆
解析：體溫過高伴隨脈搏細速為熱衰竭典型徵象，需立即物理降溫與補充電解質。⚠️ 常考重點：發燒處理與脫水評估指標。
【114 年】基本護理學
(題目：輸液時出現氣喘、紅腫熱痛現象，最可能發生？
(A)肺水腫 (B)靜脈炎或過敏反應 (C)血栓栓塞 (D)電解質失衡)
正解：(B) 靜脈炎或過敏反應
解析：注射部位紅腫熱痛為靜脈炎、氣喘與皮疹為過敏反應，需立即停藥並給予抗組織胺治療。⚠️ 常考重點：輸液治療併發症辨識與處理。
【115 年】基本護理學
(題目：糖尿病患者低血糖時最快改善症狀的處理措施為？
(A)服用葡萄糖片 (B)注射胰島素 (C)增加運動量 (D)減少飲食攝取)
正解：(A) 服用葡萄糖片
解析：低血糖時需快速提升血糖，口服含糖飲料或葡萄糖片可在 5-10 分鐘內改善症狀。⚠️ 常考重點：糖尿病急性代謝症候群處理原則。
