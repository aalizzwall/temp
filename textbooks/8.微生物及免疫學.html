🔬 主題 1: 革蘭氏染色法

核心理論：
1. 革蘭氏陽性菌 (G+) 細胞壁含厚層肽聚糖 (20-80nm)，染色後呈現紫色。
2. 革蘭氏陰性菌 (G-) 細胞壁含薄層肽聚糖 (7-8nm) + 外膜脂多糖，染色後呈現紅色。
核心理論：
3. G- 菌外膜含脂質 A，是內毒素來源；G+ 菌則主要產生胞質酶與外毒素。
🔗 邏輯鏈條 1: 感染篩檢流程
A革蘭氏陽性菌細胞壁破裂
B釋放大量游離鈣離子與脂質介質
C觸發過敏反應 (Type I Hypersensitivity)
🔗 邏輯鏈條 2: 染色結果判讀差異
A檢測目標：細菌感染類型
BG+ 菌肽聚糖含量高保留結晶紫染料
C顯微鏡下呈現藍紫色陽性結果
📝 考題回顧 · 革蘭氏染色法
1. 革蘭氏陽性菌與陰性菌的細胞壁主要差異為何？
正確答案：A. G+ 細菌肽聚糖含量高且無外膜 B. G- 細菌肽聚糖含量高且無外膜
解析：G+ 菌肽聚糖層厚 (20-80nm)，無外膜；G- 菌肽聚糖薄 (7-8nm) 且有外膜。
2. 革蘭氏染色失敗的常見原因為何？
A. 熱固定過不足導致細菌脫落 正確答案：B. 結晶紫脫色不完全 (酒精濃度過低)
解析：熱固定過度反而會使細菌脫落；脫色是關鍵步驟，需控制時間與酒精濃度 (95%)。

🛡️ 主題 2: 免疫球蛋白種類

核心理論：
1. IgG：唯一能通過胎盤，半衰期最長 (21-23 天)，主要抗體佔血清抗體 75-80%。
2. IgM：分子最大五聚體，先天性免疫第一道反應，溶血素活性最強。
3. IgA：存在於黏膜分泌液 (乳汁、唾液)，抵抗病原入侵。
4. IgE：與過敏反應相關，介在嗜酸性球表面受體結合。
5. IgD：主要作為 B 細胞受體。
🔗 邏輯鏈條 3: 新生兒抗體來源
A母體產生 IgG 抗體 (抗風疹、B 肝等)
BFcRn 載體蛋白介導轉運機制 (pH 6.0 結合→pH 7.4 釋放)
C新生兒出生時具有抗感染能力 (保護期約 6 個月)
🔗 邏輯鏈條 4: 過敏休克診斷
A接觸特定過敏原 (如蜂毒、花粉)
BIgE 與肥大細胞/FcεRI 受體結合 (致敏階段)
C再遇過敏原時釋放組織胺引發血管性水肿與休克
📝 考題回顧 · 免疫球蛋白
1. 何種免疫球蛋白是唯一能通過胎盤的抗體？
正確答案：A. IgG B. IgM C. IgA
解析：IgG 透過 FcRn 載體穿過胎盤，保護新生兒；IgM 分子量太大無法通過。
2. 過敏性休克時血清中哪類抗體最可能升高？
正確答案：A. IgE B. IgG
解析：IgE 結合肥大細胞受體，遇過敏原導致去顆粒化釋放組織胺。
3. 母體 IgG 保護新生兒的機制為何？
正確答案：A. FcRn 載體介導轉運
解析：FcRn 在酸性環境 (胎盤) 結合 IgG，中性環境釋放至血液。

🦠 主題 3: 常見病毒特性

核心理論：
1. DNA 病毒：通常在細胞核內複製 (如疱疹病毒、papillomavirus、痘病毒例外)。
2. RNA 病毒：多在細胞質內複製，變異率高 (流感病毒、HIV、柯薩奇病毒)。
3. HIV: 單股 RNA 病毒，需經逆轉錄酵素 (RT) 轉為 DNA，整合至宿主基因組。
🔗 邏輯鏈條 5: 抗病毒藥物選擇
A逆轉錄酵素 (RT) 阻斷機轉
BRNA 病毒需經過 RT 步驟合成 DNA (cDNA)
CAZT/3TC/Tenofovir 針對 HIV RNA 病毒有效
📝 考題回顧 · 病毒特性
1. HIV 病毒屬於哪一類病毒？
正確答案：A. RNA 病毒 B. DNA 病毒
解析：HIV 是單股 RNA 病毒，需經逆轉錄酵素 (RT) 轉為 DNA。
2. 下列哪一類抗病毒藥物針對 HIV 有效？
正確答案：A. 逆轉錄酵素阻斷劑 (AZT) B. DNA 聚合酶抑制劑
解析：HIV 需經 RT 步驟，故 AZT/3TC 針對此機轉。

💉 主題 4: 過敏反應分類

核心理論：
1. Type I (速發型)：IgE 介導，如過敏性鼻炎、過敏休克。
2. Type II (細胞毒型)：抗體攻擊細胞表面抗原 (如溶血性疾病、Goodpasture syndrome)。
3. Type III (免疫複合體型)：抗原抗體複合物沉積發炎組織 (如血清病、SLE)。
4. Type IV (遲延型)：T 細胞介導，如接觸性皮膚炎、移植排斥。
🔗 邏輯鏈條 6: 過敏休克急救流程
A肥大細胞釋放組織胺與白血球三烯
B血管擴張、血壓下降、喉頭水腫
C立即注射腎上腺素 (EpiPen) 阻斷反應
🔗 邏輯鏈條 7: 血清病診斷依據
A外來蛋白質 (如抗體藥物、蛇毒)
B形成 Type III 免疫複合體沉積於血管壁
C關節痛、皮疹、發燒等發炎症狀
📝 考題回顧 · 過敏反應
1. 蜂螫傷引發的過敏休克屬於哪一型？
正確答案：A. Type I B. Type II
解析：IgE 介導的速發型過敏反應。
2. 接觸性皮膚炎的免疫機轉為何？
正確答案：A. Type IV
解析：T 細胞延遲性過敏，非抗體介導。
3. 血清病的免疫機轉為何？
正確答案：A. Type III (免疫複合體)
解析：抗原抗體複合物沉積於血管壁引發發炎。

🦠 主題 5: 寄生蟲分類與特性

核心理論：
1. 原生動物 (Protozoa): 單細胞，如阿米巴、鞭毛蟲、球蟲。
2. 蠕蟲 (Helminths): 多細胞，分為扁虫 (渦蟲)、線虫、吸虫。
3. 傳播途徑：食物水源、昆蟲媒介、土壤接觸、直接人傳人。
🔗 邏輯鏈條 8: 阿米巴痢疾病理機轉
A溶组织内阿米巴侵入腸黏膜上皮細胞
B產生蛋白酶溶解組織形成假膜性潰瘍
C臨床表現：血水樣糞便、腹部絞痛
🔗 邏輯鏈條 9: 鉤蟲感染致貧機轉
A成蟲附着小腸黏膜吸血
B造成慢性鐵流失與微血管出血
C發展成缺鐵性貧血 (IDA)
📝 考題回顧 · 寄生蟲學
1. 阿米巴痢疾的糞便特徵為何？
正確答案：A. 血水樣 (果醬狀) B. 米湯樣 C. 黃白色成形糞
解析：阿米巴造成假膜性潰瘍，血液與黏液混合呈血水樣。
2. 鉤蟲感染導致貧血的機制為何？
正確答案：A. 慢性鐵流失 B. 維生素 B12 缺乏
解析：鉤蟲寄生小腸吸血，造成微血管出血與鐵流失。
3. 下列哪一類蠕蟲屬於扁虫綱？
正確答案：A. 吸虫 (如華支睾吸蟲) B. 線虫 (如蛔蟲)
解析：扁虫包括渦蟲、吸虫；線虫為圓柱形。

🍄 主題 6: 黴菌與真菌

核心理論：
1. 黴菌 (Molds): 多細胞真菌，形成菌絲與孢子。
2. 念珠菌 (Candida): 條件致病菌，常見口腔、陰道念珠菌症。
3. 曲黴菌 (Aspergillus): 空氣傳播，可形成結核性肉芽腫。
4. 黴毒：吸塵黴菌孢子後產生過敏反應。
🔗 邏輯鏈條 10: 念珠菌症誘發機轉
A廣效抗生素使用破壞正常菌群
B念珠菌過度繁殖 (條件致病原活化)
C形成口腔或陰道白色斑塊
🔗 邏輯鏈條 11: 黴菌性結核病機轉
A吸入空氣中曲黴菌孢子
B肺組織形成肉芽腫性發炎反應
CX 光呈現球形結核瘤
📝 考題回顧 · 黴菌與真菌
1. 廣效抗生素使用後最常引發的黴菌感染為何？
正確答案：A. 念珠菌症 (口腔/陰道) B. 曲黴菌肺炎
解析：正常菌群受抑制後，念珠菌成為優勢菌種。
2. 口腔念珠菌症的臨床特徵為何？
正確答案：A. 黏膜白色斑塊，刮除後有出血 B. 膿性分泌物
解析：念珠菌膜可刮除，基底面潮紅出血。

💊 主題 7: 抗生素基本原理

核心理論：
1. β-內醯胺類：抑制細胞壁合成 (青黴素、頭孢菌素)。
2. 氨基甙類：抑制蛋白質合成，腎毒性。
3. 大環內醯胺：抑制細菌核糖體，用於支原體感染。
4. 喹諾酮類：抑制 DNA 旋轉酶，影響細菌複製。
🔗 邏輯鏈條 12: 青黴素類抗菌機轉
A結合青黴素結合蛋白 (PBPs)
B抑制轉肽酶活性阻止橫向交聯
C細胞壁無法合成導致細菌裂解
🔗 邏輯鏈條 13: 氨基甙類毒性機轉
A蓄積於近曲小管細胞
B破壞線粒體與微絲組織
C導致近曲小管壞死與無尿症
📝 考題回顧 · 抗生素原理
1. 青黴素類抗生素的作用機轉為何？
正確答案：A. 抑制細胞壁合成 B. 抑制 RNA 轉錄
解析：青黴素結合 PBPs 抑制橫向交聯，阻止細胞壁成熟。
2. 下列哪一類抗生素有腎毒性？
正確答案：A. 氨基甙類 (gentamicin) B. 大環內醯胺類
解析：氨基甙易蓄積腎小管，需監測血中濃度。
3. 頭孢菌素可能引發何種過敏反應？
正確答案：A. 青黴素交叉過敏性 B. 病毒性肝炎
解析：頭孢菌素側鏈結構與青黴素有相似之處。

🔬 主題 8: 抗體反應與診斷

核心理論：
1. 直接螢光免疫法 (DFA): 標記抗體直接結合抗原。
2. 間接免疫螢光 (IFA): 檢測血清中抗體濃度。
3. ELISA: 酶聯免疫吸附測定，用於定量分析。
4. Western Blot: 確認蛋白質存在。
🔗 邏輯鏈條 14: 補體固定試驗判讀
A抗體與抗原結合形成複合物
B補體被激活並消耗
C加入指示系統後呈現陽性 (無補體剩餘)
📝 考題回顧 · 抗體診斷技術
1. 何種試驗用於確認紅血球碎片？
正確答案：A. Western Blot B. ELISA
解析：Western Blot 用於確認蛋白質存在。
2. DFA 法適用於哪一類診斷？
正確答案：A. 直接檢測抗原 (如肺炎鏈球菌) B. 間接抗體定量
解析：DFA 為直接螢光法，標記抗體結合樣本。

🦺 案例 1: 新生兒黃疸與母體抗體
情境：一名母親有紅血球不規則抗原 (如 Rh 陰性) 的病史，現在要生產。嬰兒出生後出現黃疸。

A. 母親為 Rh 陰性，父親為 Rh 陽性
B. 第一胎產生抗體時間不足，第二胎母體產生抗 Rh 抗體 (IgG)

🦺 案例 2: 銀髮族黴菌感染
情境：一位 80 歲長期臥床的糖尿病患者，足部傷口化膿且惡臭。

🧪 細菌學深入理解
3. 下列何種細菌屬於厭氧菌？
正確答案：A. Clostridium tetani (破傷風桿菌) B. E. coli C. Staphylococcus aureus
解析：Clostridium 為厭氧桿菌，需無氧環境生長。
4. 綠膿桿菌的特性為何？
正確答案：A. 產生藍綠色色素 + 壞疽味 B. 紅色色素
解析：Pseudomonas aeruginosa 為條件致病菌，常見 CF、壓瘡感染。
5. 奈瑟氏菌屬包括哪些疾病？
正確答案：A. 淋病 (N. gonorrhoeae) & 腦膜炎球菌 (N. meningitidis) B. 梅毒
解析：奈瑟氏菌為 G-雙球菌，與性傳播疾病、敗血症相關。

🛡️ 免疫學深入理解
6. 何種細胞表現 MHC Class I?
正確答案：A. 所有有核細胞 B. 僅抗原提呈細胞 (APCs)
解析：MHC I 表達於所有有核細胞，用於激活 CD8+ T 細胞。
7. 補體 C3a、C5a 的功能為何？
正確答案：A. 過敏毒素 (Anaphylatoxin)，招募中性球 B. 溶解細胞
解析：C3a、C5a 為過敏毒素，誘導發炎反應與血管擴張。
8. 細胞毒型 (Type II) 过敏反应代表疾病為何？
正確答案：A. 自身免疫性溶血性貧血 (AIHA) B. 過敏性鼻炎
解析：抗體結合 RBC 表面抗原，活化補體導致溶血。

🦠 病毒學深入理解
9. HIV 的主要致病機轉為何？
正確答案：A. 感染 CD4+ T 細胞 → 進行性免疫缺陷 B. 直接殺傷神經細胞
解析：HIV 嗜淋巴細胞 (CD4+)，導致 AIDS。
10. HPV(人類乳突病毒) 與何種疾病相關？
正確答案：A. 子宮頸癌、肛門癌 B. 肝癌
解析：HPV 感染與上皮細胞癌變相關，可預防性疫苗。

💊 藥理學深入理解
11. Vancomycin 的作用機轉為何？
正確答案：A. 結合 D-Ala-D-Ala，抑制細胞壁合成 B. 抑制蛋白質合成
解析：Vancomycin 為糖醣肽類，與青黴素作用類似。
12. 喹諾酮類的副作用為何？
正確答案：C. 肌腱斷裂、QT 延長 A. 聽力下降
解析：喹諾酮類禁忌用於兒童與懷孕 (影響骨質生長)。
13. Azithromycin 適用於哪一類感染？
正確答案：A. 支原體肺炎、衣原體感染 B. 革蘭氏陰性菌腸炎
解析：大環內醯胺為細胞內抗菌，用於無細胞壁細菌。

📋 臨床情境題
14. 一位孕婦發現有梅毒血清學陽性 (RPR 1:8)，下列處理何者正確？
正確答案：A. Penicillin G IM 每週注射一次，持續 3 週 B. Doxycycline
解析：Penicillin G 為唯一能有效過胎盤的抗生素。
15. 一位新生兒出現黃疸，母親為 Rh 陰性，嬰兒為 Rh 陽性。下列處理何者正確？
正確答案：A. 監測膽紅素水平，必要時交換輸血 B. 立即注射免疫球蛋白
解析：RhIG 在分娩後給予母親，而非新生兒。

🎓 高階綜合題
16. 關於 HIV 治療，下列敘述何者正確？
正確答案：B. HAART(高活性抗逆轉錄病毒療法)為標準治療 A. 單一抗生素即可控制 HIV
解析：HIV 為 RNA 病毒，需多種藥物抑制。

正確答案：C. 骨髓炎 (Osteomyelitis) A. 單純皮膚感染
解析：糖尿病患者 + 慢性傷口 + X 光骨頭陰影，為骨髓炎。
