微生物及免疫學 📚

🌟 核心考點總覽 (涵蓋 110-115 年國考)

本主題重點：以下知識點在護理師國家考試中頻繁出現，需特別記憶與理解！

A 革蘭氏陽菌特徵
  • 紫紅色、厚肽聚糖層（Gram + Stains：Strep, Staph, Corynebacterium）
B 革蘭氏陰菌特性
  • 紅紫色、薄肽聚糖層+外膜（Gram - Stains：E. coli, Salmonella）
C 染色解讀標準
  • +球菌成串（Strep）、+球菌成簇（Staph）、-桿菌（E.coli）
A IgG 特性功能
  • 可穿越胎盤、二次免疫反應主要抗体（占總免疫球蛋白 80%）
B IgM 特性功能
  • 一級免疫反應主要抗体、五聚體結構（分子量最大）
C IgA 特性功能
  • 黏膜表面免疫（唾液/淚液/乳汁）、分泌型 IgA（二重鏈結構）
A RNA 病毒特性
  • RNA polymerase 不依賴核糖體、易變異性（HIV, Influenza）
B DNA 病毒複製週期
  • DNA virus→整合入宿主染色體（Hepadnavirus 如 Hepatitis B）
C 病毒包膜特性
  • Lipid bilayer、親水蛋白 spike/glycoprotein、易被溶劑溶解
A Type I（速發性）過敏
  • IgE 介導、肥大細胞脫顆粒、支氣管收縮（Anaphylaxis）
B Type II（細胞毒性）反應
  • 抗體介導補體活化、紅血球溶血（Rh incompatibility）
C Type III（免疫複合體）
  • 抗原 - 抗体複合物沉積、血清病、系统性紅斑性狼瘡
A Protozoa（原生動物）
  • Toxoplasma gondii（貓糞便）、Giardia lamblia（水源性腹瀉）
B Helminths（蠕蟲）治療
  • Metronidazole（原生動物）、Albendazole、Ivermectin（線蟲）
C 寄生虫檢體採集
  • 糞便直接塗抹法、濃縮浮游法（提高診斷率）
A Candida（酵母型）
  • Nystatin, Fluconazole、念珠菌血症需血培養確認
B Aspergillus（絲狀型）
  • Voriconazole、曲黴菌肺炎（免疫低下族群易感染）
C Cryptococcus 特性
  • 膠囊構造（抗吞噬）、墨汁染色顯微鏡檢查
A Beta-lactams 類
  • Inhibit cell wall synthesis（Penicillin, Cephalosporin）
B Macrolides 類作用
  • Inhibit protein synthesis（bind 50S ribosomal subunit）
C Aminoglycosides 特性
  • Inhibit protein synthesis、耳腎毒性風險高、不可合併使用
A Live attenuated vaccine
  • M. rubella, M. varicella（可產生持續免疫）
B Inactivated vaccine
  • Influenza injectable, Hepatitis A/B、需加強劑
C Conjugate vaccine
  • Pneumococcal conjugate（PSV）用於幼兒族群
A ELISA 檢驗原理
  • 抗原-抗体結合反應、酵素標記讀數（OD 值定量）
B Western Blotting
  • 蛋白質分離技術（分離特定分子）
C Flow Cytometry 特性
  • 免疫螢光染色檢測、細胞表面抗原分析
A 接觸隔離措施
  • 一次性手套/加護衣、專用設備、標記隔離標籤
B 飛沫隔離措施
  • 外科口罩（3 層構造）、病人面向門側安置
C 空氣隔離措施
  • N95/FFP2 級別防護、負壓隔離病房設置
A Horizontal gene transfer
  • 轉化（uptake DNA）、接合（plasmid transfer）
B Target modification
  • Rifampin 抗性菌產生 new target sequence
C Efflux pump mechanism
  • Pump antibiotic out、降低藥物濃度（多重抗藥性）

🔍 考題回顧 (110-115 年精選)


【110 年】微生物及免疫學
(題目：革蘭氏染色時，陰性菌呈現何種顏色？
(A)紫色 (B)紅色 (C)藍色 (D)綠色)
正解：(B) 紅色
解析：Gram - Stains（負性）因薄壁肽聚糖層+外膜脫色，結晶紫染料被洗除，Safranin 染色呈紅色。⚠️ 常考重點：革蘭氏染色解讀標準與臨床應用。
【111 年】微生物及免疫學
(題目：下列何者為 IgE 主要功能？
(A)清除抗原 (B)過敏反應（速發性）(C)抗體介導細胞毒 (D)抗體介導補體活化)
正解：(B) 過敏反應（速發性）
解析：IgE 主要結合肥大細胞與嗜鹼性白血球，導致脫顆粒與速發性過敏反應。⚠️ 常考重點：免疫球蛋白分類與臨床特性。
【112 年】微生物及免疫學
(題目：曲黴菌感染首選治療藥物為？
(A)Amphotericin B (B)Fluconazole (C)Voriconazole (D)Echinocandin)
正解：(C) Voriconazole
解析：Voriconazole 為曲黴菌感染第一線治療藥物，具有廣譜抗黴菌特性。⚠️ 常考重點：抗黴菌藥物分類與適應症選擇。
【113 年】微生物及免疫學
(題目：Type I 過敏反應之主要介質為？
(A)IgG (B)IgM (C)IgA (D)IgE)
正解：(D) IgE
解析：Type I（速發性）過敏反應由 IgE 介導，結合肥大細胞 FcεRI 受體引發脫顆粒。⚠️ 常考重點：過敏反應分類與免疫機制。
【114 年】微生物及免疫學
(題目：下列何者為細菌多重抗藥性（MDR）主要形成機制？
(A)細胞膜通透性改變 (B)抗生素分解酵素產生 (C)藥物外排泵過表達 (D)以上皆是)
正解：(D) 以上皆是
解析：MDR 菌體產生三種抗性機制（通透性降低、酵素分解、外排泵）導致抗藥性。⚠️ 常考重點：抗微生物藥物抗性機制與感染控制策略。
【115 年】微生物及免疫學
(題目：墨汁染色可用於診斷何種黴菌感染？
(A)念珠菌 (B)隱球菌 (C)毛黴菌 (D)曲黴菌)
正解：(B) 隱球菌
解析：Cryptococcus neoformans 具膠囊結構，墨汁染色可顯微鏡下觀察膠囊構造。⚠️ 常考重點：黴菌診斷技術與鑑別特性。
💡 記憶法則：
A(分類/特性)→B(診斷方法)→C(治療策略)
⚠️ 常考：革蘭氏染色、免疫球蛋白、病毒複製三大主題
📌 考試重點：微生物及免疫學涵蓋病原體分類、免疫反應機制與感染控制三大領域，建議熟悉診斷技術與治療藥物！
