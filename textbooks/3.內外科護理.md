## 🔴 **Chapter 1: 知覺剝削與知覺繁增**  

### 📌 **核心概念**
- **知覺 (Perception)**：感官輸入 → 大腦處理 → 形成意義的過程  
- **知覺剝削 (Perceptual Deprivation)**：過度暴露於單調刺激導致知覺功能退化  
- **知覺繁增 (Perceptual Cluttering)**：過量雜訊干擾導致注意力分散  

### 🧠 **國考記憶法則**
> 💡 **"感官輸入 ≠ 知覺形成"**  
> - 感覺（Sensation）= 接收刺激 → 知覺（Perception）= 大腦解讀刺激  
> - **關鍵差異**：同一刺激可被不同個體解讀為不同意義  

### 📋 **臨床情境分析題**

**案例 A**:  
> 一位 ICU 病人接受 72 小時無干擾治療，環境安靜且無視覺刺激。護理師發現患者開始出現幻覺、躁動不安。此現象最可能屬於？

A. 知覺剝削症狀  
B. 知覺繁增症狀  
C. 正常適應反應  
D. 藥物戒斷表現  

**解答與解析**：  
✅ **答案 A**：持續無干擾治療導致大腦缺乏必要刺激 → 知覺剝削症狀  
- **考點**：ICU 病人需定期提供感官刺激（視聽音樂、家人訪視、自然光）  
- **記憶要點**：「安靜環境」+「長期無刺激」= 剝削風險

## 🔴 **Chapter 2: 疼痛病人之護理**  

### 📌 **核心概念**
```
疼痛分類架構：
├─ 急性疼痛（<3 個月）→ 組織傷害或潛在傷害
├─ 慢性疼痛（>3 個月）→ 神經病理性/軀體性/混合性
└─ 爆發性疼痛（Breakthrough Pain）→ 突破性疼痛控制失敗
```

### 📝 **國考記憶法則**
> 💡 **"疼痛四階段評估模型"**  
> 1. **辨識 (Identify)**：部位、性質、強度（0-10 分）  
> 2. **評估 (Evaluate)**：NRS 量表/數詞描繪法  
> 3. **介入 (Intervene)**：藥物・非藥物・多模式止痛  
> 4. **再評估 (Re-evaluate)**：每 30-60 分鐘監測效果

## 🔴 **Chapter 3: 免疫系統疾患之護理**  

### 📌 **核心概念**
```
免疫系統失衡三類主要病理：
├─ 過敏反應（Hypersensitivity）→ 免疫過度活躍
│   └─ Type I~IV：I型速發/II型細胞毒/III型免疫複體/IV型延遲
├─ 自體免疫疾病（Autoimmune）→ 攻擊自身抗原
│   └─ SLE、類風濕性關節炎、Type 1 DM、重症肌無力
└─ 免疫缺陷疾病（Immunodeficiency）→ 免疫機能低下
    └─ 先天性/獲得性（AIDS）、藥物性抑制
```

### 🧠 **國考記憶法則**
> 💡 **"過敏反應 Type I~IV 速記口訣"**  
> **「一二急死，四慢」**  
> - **Type 1**：急發（Anaphylaxis）- IgE 介導，秒級反應  
> - **Type 2**：致死（Cytotoxicity）- 細胞毒作用（如: TTP）  
> - **Type 3**：免疫複體（Immune Complex）- 沉積致發炎（如: SLE）  
> - **Type 4**：延遲（Delayed）- T 細胞介導，48-72h 出現

**案例 B**:  
> 32 歲女性，近半年出現晨起僵硬、關節腫脹，抗 CCP 陽性。此患者最可能患有？

A. 類風濕性關節炎（RA）  
B. 全身性紅斑狼瘡（SLE）

- **記憶要點**：「晨起僵硬」+「對稱性腫脹」= 自體免疫關節炎

## 🔴 **Chapter 4: 癌症病人之一般護理**  

### 📌 **核心概念**
```
癌症治療三支柱架構：
├─ 手術治療（Surgical）→ 局部病灶移除
├─ 化學治療（Chemotherapy）→ 全身性細胞毒藥
│   └─ 分類：細胞毒殺劑/抗代謝物/烷化劑/抗生素類似物
├─ 放射線治療（Radiation）→ 局部 DNA 損傷
└─ 整合照護（Integrative Nursing）
    ├─ 疼痛管理
    ├─ 副作用監測
    ├─ 心理社會支持
    └─ 復健康復計畫
```

### 🧠 **國考記憶法則**
> 💡 **"化學治療副作用「五字訣」"**  
> **「掉、吐、痛、疲、感」**  
> - **掉**：脫髮（Hair loss）→ 毛囊細胞毒殺  
> - **吐**：噁心嘔吐（N/V）→ GI 黏膜損傷  
> - **痛**：骨髓抑制導致的疼痛感染風險  
> - **疲**：疲倦無力（Fatigue）→ 代謝需求增加  
> - **感**：知覺異常、神經毒性（Neuropathy）

## 🔴 **Chapter 5: 體液、電解質與酸鹼不平衡之護理**  

### 📌 **核心概念**
```
體液失衡三大類：
├─ 缺水（Dehydration）→ ECF + ICF 同時減少
│   └─ 分型：等張性/高張性/低張性脫水
├─ 水過多（Overhydration）→ ECF + ICF 同時增加
│   └─ 分型：水中毒、肺水腫
└─ 電解質失衡 → Na+/K+/Ca2+/Mg2+等離子濃度異常
    └─ 各引發獨特臨床表現
```

### 🧠 **國考記憶法則**
> 💡 **"脫水分型判斷口訣：「看比重、測滲透壓」"**  
> - **等張性脫水**：血清 Na+ 正常（135-145 mEq/L）→ ECF 與 ICF 同比例減少  
> - **高張性脫水**：血清 Na+ >145 → 細胞內液流失（口渴感明顯）  
> - **低張性脫水**：血清 Na+ <135 → 細胞內液擴張（浮腫、低血壓）

### ⚠️ **臨床情境分析題**

**案例 D**:  
> 45 歲女性因嚴重嘔吐導致脫水，檢查結果顯示：Na+ = 150 mEq/L, K+ = 3.2 mEq/L。此病人最可能面臨的緊急併發症為何？

A. 低血鉀導致的心律不整  
B. 低鈉導致的意識混亂  
C. 代謝性鹼中毒導致的抽搐  
D. 高滲透壓導致的水中毒  

**解答與解析**：  
✅ **答案 A**：K+ = 3.2 <3.5 → 輕度低血鉀，嘔吐會丟失 HCl（H+ + Cl-）導致代謝性鹼中毒，但最緊急的併發症是心臟問題

- **記憶要點**：「低血鉀=心律風險」、嘔吐丟 K+ + HCl → 代謝性鹼中毒

## 🔴 **Chapter 6: 傳染病病人之護理**

```\

> 💡 **口訣「色碼防疫」**：紅觸橙飛藍空

## 🔴 **Chapter 7: 急症病人之護理**

### 📌 **核心概念**
```\
急性病變「黄金時間」原則：
├─ 腦中風（Stroke）→ 3-4.5 小時內給藥（tPA）
├─ 心肌梗死（MI）→ 90 分鐘內再通血管
└─ 外傷性休克 → ABCDE 評估流程
```

### 🧠 **國考記憶法則**
> 💡 **「ABCDE 急救評估順序」**  
> - **A**irway（氣道）→ 保持呼吸道暢通

## 🔴 **Chapter 16: 血液系統疾患之護理**

### 📌 **核心概念**
```
血液疾病三大分類：
├─ 紅血球疾病 → 貧血 (Anemia)、紅血球增多症
│   └─ 分類：缺鐵性 / 惡性 (B12缺乏) / 溶血性 / 再生不良性
├─ 白血球疾病 → 白血病 (Leukemia)、中性球減少症
└─ 血小板與凝血疾病 → 血小板低下 (Thrombocytopenia)、血友病 (Hemophilia)
```

### 🧠 **國考記憶法則**
> 💡 **「貧血症狀通用口訣：三低一快」**
> - **低**血紅蛋白 $\rightarrow$ 組織缺氧 $\rightarrow$ 皮膚蒼白 / 黏膜蒼白
> - **低**耐力 $\rightarrow$ 活動後呼吸困難 / 易疲勞
> - **低**血壓 (嚴重時) $\rightarrow$ 心輸出量下降
> - **快**心率 $\rightarrow$ 補償性心跳加快 (Tachycardia) 以維持灌流

|----------|----------|----------|----------|

> B. 給予解熱劑並繼續觀察

✅ **答案 C**：腰痛 + 寒顫 + 血壓下降 $\rightarrow$ 典型急性溶血反應 (Acute Hemolytic Reaction)。
- **考點**：任何疑似輸血反應的首要行動永遠是「立即停止輸血」，防止更多抗原進入體內。
- **記憶要點**：「腰痛 $\rightarrow$ 溶血 $\rightarrow$ 停血」。

## 🔴 **Chapter 17: 神經系統疾患之護理**

### 📌 **核心概念**
```
神經急症監控核心：
├─ 意識狀態 $\rightarrow$ GCS (Glasgow Coma Scale) 評分
├─ 顱內壓 (ICP) $\rightarrow$ 庫欣氏三聯徵 (Cushing's Triad)
└─ 中風 (Stroke) $\rightarrow$ 缺血性 (Ischemic) vs 出血性 (Hemorrhagic)
```

### 🧠 **國考記憶法則**
> 💡 **「Cushing's Triad (庫欣氏三聯徵) = 腦壓爆表」**
> - **BP $\uparrow$** (收縮壓升高，脈壓差 widen) $\rightarrow$ 身體嘗試將血液壓入高壓腦室
> - **HR $\downarrow$** (心搏過緩) $\rightarrow$ 對於血壓升高的反射性反應
> - **Resp $\downarrow$ / Irregular** (呼吸不規則) $\rightarrow$ 腦幹壓迫導致呼吸中樞失效
> **口訣：「壓高、心慢、息亂」**

### 📋 **中風 (Stroke) 護理重點**
| 項目 | 缺血性中風 (Ischemic) | 出血性中風 (Hemorrhagic) |
|------|-------------------|-------------------|
| 病理 | 血管堵塞 (血栓/栓塞) | 血管破裂 (出血) |
| 首選治療 | tPA (溶栓藥) $\rightarrow$ 3-4.5h 窗期 | 降壓、手術減壓、止血 |

### ⚠️ **臨床情境分析題**
**案例 L**:
> 55 歲男性突發右側肢體無力及言語不清，送至急診時 GCS 為 E3 V3 M5。血壓 180/100 mmHg，心率 55 bpm，呼吸 10 次/分且不規則。此患者最可能出現之併發症為何？
> 
> A. 肺栓塞
> B. 顱內壓 (ICP) 嚴重升高
> C. 心肌梗死
> D. 低血容量休克

**解答與解析**：
✅ **答案 B**：血壓高 + 心率慢 + 呼吸不規則 $\rightarrow$ 完全符合 Cushing's Triad $\rightarrow$ 提示腦疝 (Brain Herniation) 風險極高。
- **考點**：辨識 ICP 升高的晚期徵象。
- **記憶要點**：「壓高心慢息亂 $\rightarrow$ 腦壓爆表」。

## 🔴 **Chapter 18: 肌肉骨骼系統疾患之護理**

### 📌 **核心概念**
```
骨科急症與照護：
├─ 骨折併發症 → 腔室症候群 (Compartment Syndrome)、脂肪栓塞 (Fat Embolism)
├─ 牽引 (Traction) $\rightarrow$ 皮牽引 (Skin) vs 骨牽引 (Skeletal)
└─ 石膏 (Cast) 照護 $\rightarrow$ 神經血管評估 (Neurovascular Assessment)
```

### 🧠 **國考記憶法則**
> 💡 **「腔室症候群 5P 評估法」**
> - **P**ain (疼痛)：劇烈疼痛且不隨藥物緩解 (Earliest sign)
> - **P**allor (蒼白)：皮膚蒼白、填充時間 $\uparrow$
> - **P**aresthesia (感覺異常)：麻木、刺痛感
> - **P**ulselessness (無脈搏)：遠端脈搏消失 (Late sign)
> - **P**aralysis (癱瘓)：無法主動活動
> **口訣：「痛、白、麻、無、癱」**

### 📋 **牽引與石膏護理對照表**
| 項目 | 皮牽引 (Skin) | 骨牽引 (Skeletal) | 石膏 (Cast) 照護 |
|------|----------|----------|----------|

> 
> A. 增加止痛藥劑量
> B. 將患肢抬高於心臟水平以上
> C. 立即通知醫師，準備進行筋膜切開術 (Fasciotomy)
> D. 檢查石膏是否過緊，並將其稍微剪開

✅ **答案 C**：劇痛 $\neq$ 止痛藥 $\rightarrow$ 感覺異常 $\rightarrow$ 典型腔室症候群 (Compartment Syndrome)。
- **考點**：腔室症候群是外科急症，延誤處理會導致肌肉壞死。
- **記憶要點**：「止痛無效 + 麻木 = 筋膜切開」。

## 🔴 **Chapter 19: 眼耳鼻喉科疾患之護理**

## 🔴 **Chapter 20: 老年人及瀕死病人之護理**

### 📌 **核心概念**
```
終末期照護核心：
├─ 安寧療護 (Hospice/Palliative Care) $\rightarrow$ 舒適優先於治癒
├─ 哀傷過程 $\rightarrow$ Kubler-Ross 五階段
└─ 法律倫理 $\rightarrow$ DNR (Do Not Resuscitate) / 預立醫療決定 (AD)
```

### 🧠 **國考記憶法則**
> 💡 **「Kubler-Ross 哀傷五階段」**
> **「否、怒、協、鬱、接」**
> 1. **否認 (Denial)**：「這不可能，檢查一定錯了」
> 2. **憤怒 (Anger)**：「為什麼是我？這不公平！」
> 3. **討價還價 (Bargaining)**：「只要我能活到孩子結婚，我願意...」
> 4. **憂鬱 (Depression)**：「一切都沒意義了，我好累」
> 5. **接受 (Acceptance)**：「我知道時間到了，我想平靜地離開」

| 神經 | 意識模糊、幻覺、昏睡 | 安靜環境、溫柔陪伴、允許家屬在側 |

### ⚠️ **臨床情境分析題**
**案例 O**:
> 80 歲末期癌症患者已簽署 DNR 單。在瀕死階段，患者出現明顯的「瀕死喘鳴 (Death Rattle)」且家屬感到非常焦慮。護理師應如何處理？
> 
> A. 立即執行深層吸痰以清除所有痰液
> B. 給予高流量氧氣治療以改善血氧
> C. 將患者調整為側臥位，並向家屬解釋這是自然的瀕死過程
> D. 強制執行 CPR 以維持呼吸道暢通

**解答與解析**：
✅ **答案 C**：瀕死喘鳴是由於肌肉無力無法咳出分泌物，深層吸痰會造成病人痛苦且效果有限。
- **考點**：安寧療護的舒適原則 $\rightarrow$ 減少侵入性處置。
- **記憶要點**：「瀕死喘鳴 $\rightarrow$ 側臥 + 心理支持 $\rightarrow$ 舒適優先」。
