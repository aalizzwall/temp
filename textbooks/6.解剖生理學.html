解剖生理學 📚

🌟 核心考點總覽 (涵蓋 110-115 年國考)

本主題重點：以下知識點在護理師國家考試中頻繁出現，需特別記憶與理解！

A 心排血量計算公式
  • CO=SV×HR（每搏輸出量×心率，正常成人 5-6L/min）
B 心臟收縮期功能評估
  • LVEF＜40%為心衰竭指標、TAPSE＞17mm 為正常右心室功能
C 血液動力學監測指標
  • CVP（中心靜脈壓）、PCWP（肺毛細血管楔壓）、PAOP
A 肺泡換氣量計算
  • 潮氣量 (TV) - 死腔氣量 (VD)×呼吸頻率 (f)=有效換氣量
B 呼吸控制中樞定位
  • 延髓呼吸調節中心、橋腦呼吸群（吸氣與呼氣切換）
C 氧合解離曲線特點
  • HbO₂饱和度＞95%為正常、pH↑/PCO₂↓使曲線左移（Bohr 效應）
A 食道下端高壓區機制
  • 膈肌食管裂孔肌肉、食管下段括約肌（LES）壓力維持
B 胃排空速率比較
  • 碳水化合物＞脂肪＞蛋白質（脂肪延遲胃排空）
C 膽汁排泄規律
  • 進食後胆囊收縮、Oddi 括約肌鬆弛（肝胰乳頭部壓力增加）
A 腎小球濾過率 (GFR)
  • 正常成人：100-120ml/min、Creatinine clearance 為指標
B 腎小管再吸收功能
  • 近曲小管重吸收 Na⁺/H₂O（65-70%）、Loop of Henle 逆流倍增機制
C 尿液生成調節因素
  • ANP、BNP（心房利尿因子）、ADH、醛固酮、RAAS 系統
A 腦幹各區功能分佈
  • 中brain（網狀活化系統）、橋腦、延髓（心臟中樞、呼吸調控）
B 皮質脊髓束途徑
  • 運動皮層→錐體束（內囊膝部交叉至對側脊髓前角）
C 小腦平衡協調機制
  • 前庭核聯絡、深部感覺回饋迴路整合（proprioception）
A 下視丘 - 腦下垂體軸線
  • TRH-TSH-甲狀腺素、CRH-ACTH-副腎皮質激素（HPA 軸）
B 視丘下部分泌調節
  • 催乳激素抑制因子（PIF）、GnRH（促性腺激素釋放激素）
C 胰臟雙重內分泌功能
  • 外分泌：消化酵素（胰液）、内分泌：Insulin/Glucagon
A 先天與適應性免疫
  • 第一道防線（皮膚/黏膜）、第二道（吞噬細胞）、第三道（T/B 細胞）
B 抗原呈遞通路
  • APC（樹突狀細胞）→淋巴結→初級淋巴濾泡活化
C 細胞免疫與抗体免疫
  • Th1（MHC-II、CD4⁺輔助 T 細胞）、CTL（細胞毒殺性 T 細胞）
A 女性排卵週期節律
  • 卵泡期 (follicular phase)→黃體期 (luteal phase)（28 天週期）
B 男性睪丸精巢功能
  • 生精小管支持細胞、間質細胞分泌睾固酮（LH/TSH）
C 性荷爾蒙影響生理
  • 雌激素（乳腺發育）、孕激素（子宮內膜增生）
A 肌肉收縮滑動理論
  • 肌漿網 Ca²⁺釋放→actin-myosin 交聯→ATP 水解供能
B 有氧運動適應變化
  • 心肌肥大、線粒體密度增加、V₀₂max提升
C 乳酸代謝途徑
  • Muscle：Cori cycle（肌糖元→乳酸）
A 視覺光線感受機制
  • 錐細胞（明視色覺）、桿細胞（暗視黑白）→神經節細胞→視束
B 聽覺頻率分析原理
  • 基底膜不同部位振動頻率、毛細胞轉化為電訊號
C 味覺與嗅覺整合
  • 味蕾信號→三叉神經（甜苦鹹酸）、嗅上皮→嗅丘腦

🔍 考題回顧 (110-115 年精選)


【110 年】解剖生理學
(題目：心衰竭時下列何者減少？
(A)CVP (B)LVEF (C)PCWP (D)每搏輸出量)
正解：(B) LVEF
解析：LVEF＜40%為左心室射血分數降低，顯示心臟收縮功能減退。⚠️ 常考重點：心臟機能評估與血液動力學監測指標。
【111 年】解剖生理學
(題目：肺泡氧張力約為多少 mmHg？
(A)50 (B)80 (C)100 (D)40)
正解：(C) 100
解析：海平面大氣壓力下，肺泡內氧分壓約為 100mmHg（PAO₂）。⚠️ 常考重點：氣體交換機制與血氧張力關係。
【112 年】解剖生理學
(題目：正常成人腎小球濾過率 (GFR) 約為？
(A)50-60ml/min (B)70-80ml/min (C)100-120ml/min (D)150ml/min)
正解：(C) 100-120ml/min
解析：正常腎功能 GFR 為 100-120ml/min，是腎臟功能的基礎指標。⚠️ 常考重點：泌尿系統功能評估與腎衰竭監測標準。
【113 年】解剖生理學
(題目：延髓受損最可能影響何種生理功能？
(A)視覺 (B)呼吸控制 (C)膝跳反射 (D)皮膚觸覺)
正解：(B) 呼吸控制
解析：延髓含心臟中樞、呼吸調控中心，受損會導致呼吸暫停或異常。⚠️ 常考重點：神經系統解剖定位與功能關聯。
【114 年】解剖生理學
(題目：下列何者為副甲狀腺激素主要作用？
(A)抑制鈣吸收 (B)促進尿鈣排出 (C)增加血中濃度 (D)減少胃酸分泌)
正解：(C) 增加血中濃度
解析：PTH（副甲狀腺素）促進骨質溶解、腎臟重吸收鈣、腸道鈣吸收（Vit D₃活化）。⚠️ 常考重點：內分泌激素生理功能與臨床相關疾病。
【115 年】解剖生理學
(題目：G-6-PD 缺乏症易導致何種血液改變？
(A)白血球增多 (B)血小板減少 (C)溶血性贫血 (D)凝血功能障礙)
正解：(C) 溶血性貧血
解析：G-6-PD 缺乏時無法產生足夠 NADPH，過氧化物酶體無法還原活性氧（ROS），導致紅血球膜脂質过氧化、破裂溶血。⚠️ 常考重點：代謝疾病與生理機能關聯性。
💡 記憶法則：
A(器官/結構)→B(生理機制)→C(功能指標)
⚠️ 常考：循環、呼吸、腎臟功能三大系統評估
📌 考試重點：解剖生理學是基礎醫學核心，涵蓋循環、呼吸、泌尿、神經等多個系統的生理機制！
