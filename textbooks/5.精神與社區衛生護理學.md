## 🌟 一、精神疾病護理 (High Yield)

### 1.1 精神分裂症 (Schizophrenia)

#### 🌟 核心考點

**【診斷要點】**
- **Flesch Reading Ease**: 7.2 (中等難度)
- **Key Symptoms**:
  - 阳性症状：幻聽、幻覺、妄想、思维混亂
  - 阴性症状：情感平淡、社交退縮、意志缺乏
- **病程**: 至少持續 6 個月才算慢性化

**【藥物治療】💊**
```
第一代抗精神病藥 (典型)：
├── Haldol (Haloperidol) — 首選急性期躁動
├── Thorazine (Chlorpromazine) — 嗜睡副作用大
└── Trilafon (Fluphenazine) — 半衰期長

第二代抗精神病藥 (非典型)：
├── Risperdal (Risperidone) — 首選，代謝副作用小
├── Seroquel (Quetiapine) — 嗜睡、低血壓風險
├── Zyprexa (Olanzapine) — 體重增加、糖尿病風險高
└── Geodon (Ziprasidone) — QTc 延長風險，需監測

💡 高頻考點：
- Risperdal 是唯一代謝副作用較小的非典型藥物
- Clozapine 用於治療抵抗型精神分裂症（必須定期測白血球）
```

**【副作用記憶法則】** 🧠
```
「5D+1H」：
├── Dry (乾) — 流體分泌減少
├── Dizzy (暈) — 低血壓
├── Dysphagia (吞嚥困難) — 自主神經障礙
├── Depressed Mood (情緒低落) — 內臟性幻聽後的情緒問題
├── Diabetes (糖尿病) — 體重增加/代謝綜合症
└── Hyperprolactinemia (高泌乳激素) — 溢乳/陽萎

📝 臨床警示：老年人使用第一代藥物需特別注意「Dizziness + Dry」，容易跌倒！
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **自殺風險評估**
   - 🧠 **重點記憶**：精神分裂症病人**病情穩定後反而較易自殺**（急性期以被害妄想為主）
   - ⏰ **觀察指標**：治療中藥效過渡期最危險

2. **藥源性肺炎風險**
   - 🩺 長期服用抗精神病藥者，需定期檢查肺部功能
   - 💊 Olanzapine、Clozapine 較高風險

3. **代謝綜合症監測**
   ```
   Zyprexa + Seroquel → 高風險群體
   └── 需追蹤：體重/BMI/血糖/血脂/HOMA-IR
   ```

4. **藥物相互作用**
   - 🔄 CYP450 系統代謝影響：
     - Olanzapine (CYP1A2, 2D6) → 與 SSRIs 合併使用需小心
     - Quetiapine (3A4) → 與 Statins 有相互作用

#### ❌ 易錯點

- **❌ 錯誤觀念**：「精神分裂症病人需要隔離治療」
  - ✅ **正確認識**：社會功能恢復才是治療目標，應鼓勵社區融合

- **❌ 錯誤觀念**：「所有抗精神病藥都會導致體重增加」
  - ✅ **正確認識**：Risperidone、Ziprasidone 的風險較低

### 1.2 雙相情緒障礙 (Bipolar Disorder)

#### 🌟 核心考點

**【疾病分期】**
```
躁狂期 → 混合型發作 → 輕度躁狂/輕躁 → 抑鬱發作 → 快速循環型
     ↑      ↑           ↑        ↑            ↑
┌─────┼─────┬───────────┴────────┼─────────┐
│ 躁狂期：情緒高昂、活動增加、睡眠需求減少
├─────┤                            │        │
│混合型：躁狂與抑鬱症狀同時存在
└─────┘                            │        │
                                   │        │
```

**【重要記憶點】：雙相 disorder 最常併發的是「快速循環型」，需長期維持治療**

#### 🧠 **【國考記憶法則加強版】** 

**💡 「心境穩定劑選擇策略」** 💊

| 藥物 | 適用情況 | ⚠️ 注意事項 | 💡 記憶法則 |
|------|---------|-----------|-----------|
| **Lithium** | 首選，預防自殺風險最高 | 腎毒性/甲狀腺問題 | «Serum Level + TSH» |
| **Valproate** | ⚠️ 女性育齡期避免，致畸性高 | 肝毒性/血小板降低 | «Val-Prop → Watch Liver!» |
| **Carbamazepine** | - | 肝毒性需監測 | «Carb → Carbuncle (Liver)» |
| **Lamotrigine** | 抗抑鬱效果好，但啟動要慢 | 史蒂文斯強生綜合症風險 | «Lamo → Lamontagne (Slow Start)» |

### 1.3 憂鬱症 (Major Depressive Disorder)

#### 🌟 核心考點

**【診斷要點】**
- **DSM-5 Criteria**: 至少 5 項症狀，包括：
  - 情緒低落或興趣丧失（2 項必須）
  - 睡眠改變
  - 食慾改變
  - 疲勞/精力不足
  - 自我貶低/罪惡感
  - 思考能力下降
  - 自傷/自殺意念

**【藥物選擇策略】💊**
```
SSRIs (首選):
├── Sertraline (Zoloft) — 首選，適合合併焦慮
├── Escitalopram (Lexapro) — 副作用最小，老年人首選
└── Citalopram (Celexa) — QTc 延長風險

SNRIs:
├── Venlafaxine (Effexor) — 高劑量才有 SNRI 效果
└── Duloxetine (Cymbalta) — 適合合併慢性疼痛

Bupropion (Wellbutrin):
└── 不影響性功能，適合合併吸煙癮斷戒需求

📝 臨床警示：抗抑鬱藥物需「啟動期 (2-4 週)」才見效，避免病人中途中斷！
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **自殺風險評估**
   - 🧠 **重點記憶**：治療初期（第 2-4 週）情緒改善前，自殺意念可能反而增強！
   - 📋 **監測指標**：每週一次自殺風險評估表 (SISSI)

2. **藥物停用症候群**
   ```
   Paroxetine + Sertraline → 高風險（需慢慢減量）
   Fluoxetine + Bupropion → 較低風險（長半衰期）
   ```

3. **血清素症候群警訊** 🚨
   ```
   Triad Symptoms:
   ├── 自主神經興奮：心悸、發汗、震顫
   ├── 精神狀態改變：焦慮、意識混亂
   └── 肌肉超活化：肌攣、反射亢進
   
   💡 處理：停藥 + 支援性護理，嚴重者需 ICU 觀察
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「抗抑鬱藥物沒有成癮問題」
  - ✅ **正確認識**：雖無苯二氮芪類的依賴性，但停藥仍需慢慢減量

- **❌ 錯誤觀念**：「所有憂鬱症都需要抗鬱劑治療」
  - ✅ **正確認識**：輕度憂鬱可用心理治療 + 生活型態調整

## 🌟 二、社區衛生護理 (High Yield)

### 2.1 傳染病社區照護 🦠

#### 🌟 核心考點

**【高優先級傳染病】**
```
HIV/AIDS:
├── PEP (暴露後預防): 72 小時內開始，服用 4 週
├── PrEP (暴露前預防): 每月檢查 HIV RNA + 腎功能
└── ART (抗逆轉錄病毒治療): 三種藥物組合

📝 臨床警示：HIV+ 孕婦可透過 C-section 降低垂直傳染率至 <1%!

Tuberculosis (TB):
├── Initial Phase: RIPE (Rifampin, Isoniazid, Pyrazinamide, Ethambutol)
├── Continuation Phase: RH (Rifampin, Isoniazid)
└── 監測指標：肝功能 + 視神經（Ethambutol）

📝 臨床警示：異物性結膜炎需隔離治療，避免傳染他人！
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **抗逆轉錄病毒藥物副作用**
   ```
   Integrase Inhibitors (首選):
   ├── Dolutegravir — 最常見，副作用小
   └── Raltegravir — 懷孕安全
   
   Protease Inhibitors:
   └── 高膽固醇/胰島素抵抗風險
   
   💡 監測：每 3 個月一次完整血液檢查 + 腎功能
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「HIV 陽性者無法正常工作」
  - ✅ **正確認識**：現代 ART 治療下，病毒載量檢測不到=不傳染他人（U=U）

---

### 2.2 慢性病管理 🏙️

#### 🌟 核心考點

**【糖尿病社區照護】**
```
Medication Hierarchy:
├── Metformin (首選) — GI副作用 + B12缺乏風險
├── SGLT2 Inhibitors (新選擇) — 腎病/心臟病保護
├── GLP-1 Agonists — 體重減少效果好
└── Insulin — 終末期需使用

💡 臨床警示：SGLT2 抑制劑有酮酸中毒風險（即使血糖正常）!

【高血壓社區照護】
├── First Line: ACEI / ARB / CCB / Thiazide
├── Second Line: Combinaion Therapy (雙重治療)
└── Goal BP: <130/80 mmHg (高齡者可放宽至<140/90)

📝 臨床警示：ACEI + ARB 組合不可同時使用（腎衰竭風險）!
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **藥效相互作用監測**
   - 🔄 Metformin + IV造影劑 → 急性肾衰竭風險
   - 💊 Insulin + 感染/壓力 → 需調整劑量

2. **藥物依從性評估**
   ```
   高流失率群體：
   ├── 年輕族群（認為自己不會生病）
   └── 多重共病者（服用>5 種藥物）
   
   💡 解決方案：簡化製程 + 藥學教育
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「血糖正常就不用吃降糖藥」
  - ✅ **正確認識**：藥物是為了保護器官功能，不能隨意停藥

## 🌟 三、精神疾病社區復健 🔄

### 3.1 復健原則

#### 🌟 核心考點

**【復健階段劃分】**
```
急性期 → 過渡期 → 社區融合期 → 自我管理期
    ↑       ↑        ↓            ↓
┌─────┼─────┬────────┴──────────┐
│ 醫院治療 +                          │ 社會支持系統建立
│ 家屬心理教育                         │
└─────┘                            │
                                   │
【重點記憶】：社區復健的**核心**是「賦權（Empowerment）」!
```

#### ⚠️ 臨床警示

**【高風險情境】**
1. **藥物減量策略**
   ```
   Step-down Strategy:
   ├── 保持原藥劑量的 50% → 2-3 週
   ├── 轉換為長效製程 → 1-2 週
   └── 維持最低有效劑量 → 長期追蹤
   
   💡 臨床警示：突然停藥可能引發「撤藥症候群」!
   ```

#### ❌ 易錯點

- **❌ 錯誤觀念**：「精神疾病病人沒有工作能力」
  - ✅ **正確認識**：適度復健可恢復社會功能，需早期介入

## 🌟 四、注意缺陷過動症 (ADHD) 🧠⏱️

#### 🌟 核心考點

**【診斷要點】**
- **DSM-5 Criteria**: 至少 6 個症狀（兒童）/ 5 個症狀（成人），持續 >6 個月
- **三大類型**：
  - ✅ Combined Type (混合型) — 最常見，注意力不足 + 過動衝動
  - ⚠️ Predominantly Inattentive Type (注意缺失型) — 曾被稱為「ADD」
  - 🔥 Predominantly Hyperactive-Impulsive Type (過動衝動型) — 成人較少見

**【藥物治療】💊**
```
興奮劑類（首選）：
├── Methylphenidate (Ritalin, Concerta) — 短效/長效選擇多
│   └── 副作用：食慾不振、生長延遲、睡眠障礙
│
└── Amphetamine (Adderall, Vyvanse) — 合併使用較無效果時考慮
    └── 副作用：焦慮加重、心悸

非興奮劑類（二线）：
├── Atomoxetine (Strattera) — 首選用於有物質濫用史的病患
│   └── 起效慢（4-8 週），需監測肝功能
│
└── Guanfacine/Clonidine — 用於合併焦慮/過動症狀

💡 高頻考點：Concerta 使用 Osmotic Release System (OROS) 技術，每日一次即可！
```

#### 🧠 **【記憶法則：ADHD 藥物副作用口訣】**
```
「三個 A」：
├── Appetite Loss (食慾不振) — Methylphenidate 常見副作用
├── Agitation (焦慮/激越) — 高劑量時易出現
└── Anxiety (焦慮) — 成人 ADHD 較常合併

📝 記憶法則：「三個 A」→ 都以 A 開頭！考試快速排除選項用！
```

## 🌟 五、創傷後壓力症候群 (PTSD) 💔

#### 🌟 核心考點

**【診斷要點】**
- **DSM-5 Criteria**：需經歷創傷事件 + 四階段症狀（>1 個月）
```
Stage 1: Re-experiencing (反覆經驗)
├── Flashback (閃回) — 無法控制的侵入性回憶
├── Nightmares (惡夢) — 典型睡眠障礙
└── Avoidance (迴避) — 主動迴避與創傷相關刺激

Stage 2: Hyperarousal (過度警覺)
├── Hypervigilance (高度警戒)
├── Startle Response (驚跳反應) — 聽到爆炸聲嚇一跳
├── Sleep Disturbance (睡眠障礙)
└── Irritability (易怒/憤怒爆發)

Stage 3: Negative Changes (負面認知改變)
├── Guilt/Shame (罪惡感/羞恥感)
├── Emotional Numbing (情感麻木) — 無法感受到喜悅
└── Blaming Self/Others (自責/歸咎他人)

Stage 4: Affective Problems (情緒問題)
└── Depressive Symptoms (憂鬱症狀) — 合併憂鬱需綜合治療
```

**【藥物治療】💊**
```
第一線選擇：
├── SSRIs: Sertraline, Paroxetine — FDA 批准用於 PTSD
│   └── 記憶法則：「SER-PAR」→ Sertraline + Paroxetine
│
└── SNRIs: Venlafaxine — 二线選擇

第二線選擇（合併症使用）：
├── Prazosin (褪黑激素受體拮抗劑) — 改善噩夢/閃回
│   └── 臨床警示：需監測低血壓
│
└── PRN: Benzodiazepines (短期使用) — ⚠️ 有爭議，不建議長期使用
    └── 記憶法則：BDZ → Benzo DiAzipine（短效藥）

💡 高頻考點：PTSD 病人需同時使用「藥物 + 心理治療」效果最佳！
```

**【心理治療】** 🧠
```
證據支持的心理治療：
├── CBT (認知行為治療) — 第一線選擇
├── EMDR (眼動統合治療) — 針對創傷記憶再處理
│   └── 記憶法則：Eye Movement + Dissociation → EMDR
│
└── Prolonged Exposure Therapy (延遲暴露) — 逐步面對恐懼刺激
```

## 🌟 六、成癮問題處理 💉🍷

#### 🌟 核心考點

**【酒精依賴】🍷**

**【戒斷症狀評估】**
```
CIWA-Ar Scale (70-99 分 = 中重度戒斷):
├── Tachycardia (心動過速) — HR >100 bpm
├── Hyperhidrosis (多汗)
├── Tremor (震顫/手抖) — 經典徵象！
├── Visual/Hallucinatory Symptoms (幻視/幻覺)
│   └── 💡 記憶法則：CIWA → C-I-W-A-A-r → "See I Want A Alcohol"
│
└── Seizure Risk (癲癇風險) — 高劑量後 24-72 小時

💡 記憶法則：戒斷症狀出現時間 = 最後飲酒後 6-48 小時！
```

**【藥物治療】💊**
```
急性期：BENZODIAZEPINES (首選!)
├── Chlordiazepoxide (Librium) — 長效，適合門診使用
│   └── 記憶法則：「Chlor」→ 氯（穩定）+ 苯二氮
│
├── Lorazepam (Ativan) — ⚠️ 肝臟代謝不良者首選
│   └── Glucuronidation（不需 CYP450）
│
└── Diazepam (Valium) — ⚠️ 老年人避免（長半衰期）

維持期治療：
├── Naltrexone — 拮抗鴉片類受體，減少渴求感
│   └── 記憶法則：「Nal」→ "No Alcohol"
│
├── Acamprosate — 調節 GABA/Glutamate 平衡
│   └── 副作用少，適合長期維持
│
└── Disulfiram (Antabuse) — ⚠️ 避免酒精（雙硫醇反應！）
    └── 記憶法則：喝酒後會「吐」（Disulfiram → Dis-Suffer）
```

**【鴉片類成癮】🚬💉**

**【藥物替代療法】** 💊
```
Methadone (美沙酮):
├── ✅ 優點：長效（24-36 小時），可口服，減少針灸行為
│   └── ⚠️ 缺點：需定期監測（呼吸抑制風險）
│
Buprenorphine (丁丙諾菲):
├── ✅ Suboxone = Buprenorphine + Naloxone
│   ├── 高親和力 → 部分激動劑
│   └── Naloxone 皮下注射時誘發戒斷（防止濫用）
│       💡 記憶法ظة：「Sub-Boxone」→ "Under Box = Partial Effect"
│
└── Lozenges (舌下錠) — 適合門診使用

💡 高頻考點：Suboxone 使用方式需指導病人！(Swallow → 吞下，非注射/咀嚼)
```

**【尼古丁成癮】🚬**

**【戒煙療法】💊**
```
Nicotine Replacement Therapy (NRT):
├── Nicotine Patch (尼古丁貼片) — ⭐⭐⭐ (持續釋放)
│   └── 💡 記憶法則：「Patch → Place」→ 放置於皮膚上
│

## 🌟 七、老年性精神疾病 🦴🧠

### 7.1 老年失智症 (Dementia)

#### 🌟 核心考點

**【主要類型對比】**
```
Alzheimer's Disease:
├── 病理：β-Amyloid Plaques + Tau Tangles
├── 臨床表現：記憶力喪失（近期記憶→遠期記憶）
│   └── 💡 記憶法則：「A.D. → A forgets D」→ Alzheimer's Disease
│
Vascular Dementia:
├── 病理：血管栓塞/梗塞
│   └── 💡 記憶法則：「Va-Dem → Va (Vessel) + Dem」→ 血管問題！
│
Lewy Body Dementia:
├── 病理：α-Synuclein Lewy Bodies
├── 臨床表現：認知障礙 + 帕金森症狀（僵硬/震顫）+ 幻視
│   └── ⚠️ 高風險：神經勒勒抗精神病藥易引發 Malignant Neuroleptic Syndrome
│
Frontotemporal Dementia:
├── 臨床表現：行為改變 + 語言障礙（早期記憶力較好！）
│   └── 💡 記憶法則：「F.T.D. → Forget Temporal Lobe」→ 顳葉受損

💡 高頻考點：Alzheimer's 是最常見類型，佔老年失智症的 60-70%！
```

**【藥物治療】💊**

**膽鹼酯酶抑制劑（首選）：**
```
├── Donepezil (Aricept) — ⭐⭐⭐ 首選，每日一次，起效緩慢
│   └── 💡 記憶法則：「Done → Do Now」→ 立即開始！
│
├── Rivastigmine (Exelon) — 適合同時有帕金森症狀者
│   └── 💡 記憶法則：「Riva → River」→ Like a river flowing!
│
└── Galantamine (Razadyne) — 調節煙鹼性受體
    └── 💡 記憶法則：«Gala» → «Galaxy of Choline»

💡 副作用：流體分泌增加（Dry mouth, Nausea, Diarrhea）!
```

**NMDA 受體拮抗劑：**
```
Memantine (Ebixa):
├── 適用對象：中度至重度失智症
├── 💡 記憶法則：«Men-tal Mem» → Mental Memory Enhancement!
│
└── 💡 高頻考點：膽鹼酯酶抑制劑（Donepezil）+ NMDA 拮抗劑（Memantine）可合併使用！
```

**【臨床警示】⚠️**
```
1. 跌倒風險評估
   └── 需定期檢查：步態穩定性 + 藥物副作用（鎮靜/低血壓）

2. 藥源性認知障礙
   └── ⚠️ 避免使用：Benzodiazepines, Anticholinergics, Sedatives

3. 非典型精神科藥物使用警訊
   └── ⚠️ 老年人對 Neuroleptics 更敏感（需降低劑量 + 密切監測）
       💡 記憶法則：「Old + Drug = Bad」→ 老年人慎用药！
```

### 7.2 老年憂鬱症 (Late-Life Depression)

#### 🌟 核心考點

**【診斷要點】**
```
Geriatric Depression Scale (GDS):
├── Score >5: High risk of depression
│   └── 💡 記憶法則：「G.D.S. → Good Days Start」→ Good Score = No Depression!
│
⚠️ 診斷警訊：老年人可能表現為「躯体化症狀」而非情緒低落
├── 常見身體症狀：疼痛、疲勞、失眠、食慾改變
│   └── 💡 記憶法則：「Pain + Fatigue + Insomnia = Depression in Elderly!
```

**【藥物選擇策略】💊**
```
SSRIs (首選):
├── Sertraline — ⭐⭐⭐ 首選（副作用最小）
│   └── 💡 記憶法則：«Ser-ta» → «Ser-eneous」→ Safe!
│
├── Citalopram — ⚠️ QTc 延長風險（高劑量時），老年人需特別注意！
│
└── Escitalopram — ✅✅ 副作用最小，老年人首選！

💡 高頻考點：老年人抗抑鬱藥物起效慢（4-6 週），需耐心監測！
```

**【臨床警示】⚠️**
```
1. 自殺風險評估
   └── ⚠️ 老年人是自杀最高風險群體之一（尤其是男性）
       💡 記憶法則：「Old + Male = High Suicide Risk!»

2. 藥物停用症候群
   └── Paroxetine + Venlafaxine → 高風險，需慢慢減量

3. 電療治療 (ECT)
   └── 適用于：治療抵抗型、嚴重自杀風險、合併精神病症狀
       💡 記憶法則：「E.C.T. → Emergency Care Treatment」→ 緊急時使用！
```

## 🌟 八、社區復元服務與日間照顧 🏘️

**【社區復健模式對比】🔄**

| 模式 | 適用對象 | 主要功能 | 優點/缺點 |
|------|---------|---------|----------|

**【復健評估工具】📋**

**Functional Independence Measure (FIM):**
```
15 項指標：
├── Self-Care: Eating, Bathing, Grooming, Toileting, Dressing
├── Mobility: Transfer, Toilet Transfer, Locomotion
├── Communication: Expressive, Receptive
└── Cognition: Social Interaction, Problem Solving, Memory

💡 記憶法則：「F.I.M. → Functional Independence Measure」→ 
   Functional (功能) + Independent (獨立) + Measure (衡量)！
```

**【日間照顧服務】☀️**
```
主要功能：
├── Socialization (社交互動) — 💡 記憶法則：「Social → Society」
├── Activities of Daily Living (ADL) 訓練 — 💡 記憶法則：«ADL → Always Do Life!
├── Medication Management (藥物管理) — ⚠️ 需定期監測依從性
└── Family Support Education (家屬教育) — ✅✅ 最被低估！

💡 高頻考點：社區復元服務的核心是「賦權（Empowerment）」而非單純照顧！

### **📘 Chapter 9: 思考障礙病人的護理** (Thinking Disorders)

#### 🌟 **核心考點**

| 障礙類型 | DSM-5 分類 | 主要症狀 | 治療目標 |
|---------|-----------|---------|---------|
| **失語症 (Aphasia)** | - | 表達或理解語言障礙 | 溝通訓練、ADL 維持 |
| **失認症 (Agnosia)** | - | 感官輸入無法辨識 | 環境適應訓練 |
| **執行功能障礙** | - | 規劃/組織能力下降 | 結構化生活節奏 |

#### 💊 **【藥物治療】**

```
抗精神病藥：
├── Risperidone — 首選，認知副作用最小
├── Aripiprazole — 部分激動劑，代謝影響小
└── Olanzapine — 鎮靜效果好，但體重增加風險高

💡 國考重點：「執行功能障礙」無法單純靠藥物改善，需配合認知復健！
```

#### ⚠️ **臨床警示**

**【高風險情境】**
1. **溝通困難導致的誤解**
   - 🧠 **重點記憶**：失語症患者可能表達不清但理解力較好
   - ⏰ **觀察指標**：使用圖片、手勢輔助溝通
   - 💊 **護理措施**：確認病人理解指令（回示法：Ask them to repeat back）

2. **環境刺激過度導致混亂加重**
   - 🩺 複雜環境會加劇執行功能障礙病人的混亂
   - 📝 **記憶法則**：«Less Is More!」→ 簡單、清晰、重複

#### ❌ **易錯點**

- **❌ 錯誤觀念**：「失語症患者無法學習」
  - ✅ **正確認識**：理解力往往保留，可用圖片/手勢溝通訓練！

### **📘 Chapter 10: 情感障礙病人的護理** (Mood Disorders)

#### 🌟 **核心考點**

| 障礙類型 | 典型症狀 | 病程特徵 | 治療首選 |
|---------|---------|---------|---------|
| **重鬱症 (MDD)** | 情緒低落、興趣喪失 >2 週 | 單極性：只有抑鬱 | SSRIs + CBT |
| **躁鬱症 (BD)** | 躁狂 + 抑鬱交替發作 | 雙極性：兩極循環 | 心境穩定劑優先 |

#### 💊 **【藥物選擇策略】**

```
**MDD（重鬱症）：**
├── SSRIs: Escitalopram/Lexapro → 首選（副作用最小）
├── SNRIs: Venlafaxine → 合併慢性疼痛/焦慮時
└── Bupropion → 性功能副作用大、需提神者

**BD（躁鬱症）：**
├── Lithium → 預防自殺風險最高（腎/TSH 監測）
├── Valproate → 快速循環型首選（孕婦禁忌！）
├── Lamotrigine → 抗抑鬱效果好，但啟動要慢（SJS 風險）
└── Carbamazepine → 肝毒性需監測

💡 高頻考點：
- Lithium → TSH + Renal Function (腎+甲狀腺)
- Valproate → Teratogenic (孕婦禁用！)
- Lamotrigine → Slow Start! (SJS Risk)
```

#### ⚠️ **臨床警示**

**【高風險情境】**
1. **Lithium 中毒徵兆**
   - 📌 **記憶法則**：「LIthium + 5+ 症狀」→ LIght-headed, Incoordination, Tremor, Irritability, Nausea
   - ⚠️ **目標血濃度**：0.6-1.2 mEq/L（急性躁狂可達 1.5）
   - 💡 **國考重點**：發燒/脫水會減少腎排泄 → Lithium 中毒！

2. **Valproate 致畸性警告**
   - 🩺 所有育齡期女性患者：**計劃怀孕前需調整用藥**
   - 📝 **記憶法則**：«No Baby in the Lab!» (Lab = Liver) → 肝毒性高！

3. **快速循環型 BD 治療失敗**
   - 🧠 **重點記憶**：年發作<4 次為「單相型」，>4 次需調整策略為「心境穩定劑加強版」

#### ❌ **易錯點**

- **❌ 錯誤觀念**：「躁鬱症只需要抗抑鬱藥」
  - ✅ **正確認識**：單一抗抑鬱藥可能誘發躁狂發作！必須搭配心境穩定劑

### **📘 Chapter 11: 人格障礙病人的護理** (Personality Disorders)

#### 🌟 **核心考點**

| 類型分類 | 特徵行為 | 治療策略 | 風險重點 |
|---------|---------|---------|---------|
| **A 類：古怪型**<br>(Schizoid, Paranoid, Schizotypal) | 社交退縮、多疑、偏離常規 | 建立信任關係<br>認知行為介入 | 孤獨、自殺風險 |
| **B 類：戲劇型/混亂型**<br>(Antisocial, Borderline, Histrionic, Narcissistic) | 情緒不穩、衝動、 manipulative | DBT 技能訓練<br>結構化環境 | 自我傷害、關係破裂 |
| **C 類：焦慮型**<br>(Avoidant, Dependent, Obsessive-Compulsive PD) | 恐懼拒絕、依賴、完美主義 | 逐步暴露療法<br>支持性心理治療 | 社交焦慮、功能受損 |

#### 💊 **【藥物選擇策略】**

```
⚠️ 重要原則：人格障礙無特效藥，需对症治疗！

**B 類（邊界型人格）：**
├── Mood Stabilizers → Lithium/Valproate (情緒不穩)
├── SSRIs → Depression/焦慮緩解
└── Atypical Antipsychotics → 急性衝動控制

💡 國考重點：「Borderline PD = Self-harm Risk」→ 緊急處理流程！
```

#### ⚠️ **臨床警示**

**【高風險情境】**
1. **邊界型人格 (BPD) 自我傷害行為**
   - 🧠 **重點記憶**：非自殺企圖，而是「情緒調節失敗」
   - ⏰ **觀察指標**：刮臂痕跡、自殘工具、衝動性言語
   - 💊 **護理措施**：
     * DBT 技能訓練（情緒調節、痛苦耐受）
     * 不獎勵自我傷害行為，但不懲罰求助行為
     * 建立安全計畫（緊急联系人、安撫技巧清單）

2. **反社會人格 (ASPD) 治療抗拒**
   - 🩺 常以「利用醫療資源」为目的就診
   - 📝 **記憶法則**：«Lawyer's Attitude!» → 邊界堅定、契約清晰

#### ❌ **易錯點**

- **❌ 錯誤觀念**：「人格障礙無法治療」
  - ✅ **正確認識**：雖不能改變人格，但可改善功能（藥物 + 心理社會介入）

### **📘 Chapter 12: 其他精神科疾病的護理** (Other Psychiatric Disorders)

#### 🌟 **核心考點**

| 疾病分類 | 代表性疾病 | 治療策略 | 關鍵監測指標 |
|---------|-----------|---------|-------------|
| **妄想性疾患** | 被害妄想、嫉妒妄想等 | 抗精神病藥 + CBT | 症狀控制、社會功能 |
| **創傷後壓力症 (PTSD)** | 侵入回憶、閃回 | SSRIs/Prazosin + EMDR | 睡眠、焦慮程度 |
| **解離性疾患** | 人格分裂、身份喪失 | Benzodiazepines（急性期）→ Psychotherapy | 穩定身份認同 |

#### 💊 **【藥物選擇策略】**

```
**PTSD：**
├── SSRIs: Sertraline/PAROXETINE → 首選
├── Prazosin → 失眠/夜夢驚嚇 (Nightmares)
└── Prn Benzodiazepines → 急性焦慮（短期使用！）

💡 高頻考點：PTSD 患者對 SSRI 反應較慢，需耐心治療！

**解離性身份障礙 (DID)：**
├── Benzodiazepines → 急性期穩定
└── Antipsychotics → 幻覺/妄想控制

⚠️ 重要：避免長期使用苯二氮平類（依賴風險高）!
```

#### ⚠️ **臨床警示**

**【高風險情境】**
1. **PTSD 退伍軍人族群**
   - 🧠 **重點記憶**：戰創壓力導致 PTSD 發病率特別高
   - 💊 **國考重點**：需配合復健、職能訓練、家庭支持

2. **酒精相關精神障礙**
   - 🩺 戒斷期需 Benzodiazepines（預防痙攣）
   - 📝 **記憶法則**：«Alcohol Withdrawal = Benzos First!»

#### ❌ **易錯點**

- **❌ 錯誤觀念**：「PTSD 只是心理問題，不需藥物」
  - ✅ **正確認識**：神經生化學改變（HPA axis）→ 藥物治療是基礎！

### **📘 Chapter 13: 危機處理與意外事件的護理** (Crisis Intervention)

#### 🌟 **核心考點**

| 危機類型 | 緊急措施 | 長期介入 | 監測重點 |
|---------|---------|---------|---------|
| **自殺風險評估** | 1:1 陪伴、移除危險物品 | CBT + 藥物治療 | 情緒、計畫、手段可得性 |
| **急性精神病發作** | 制動藥（Haldol）、隔離保護 | 抗精神病藥維持治療 | 症狀控制、社會功能恢復 |
| **創傷事件後急應症** | 穩定情緒、安全環境 | EMDR + PTSD 心理療法 | 睡眠、閃回頻率 |

#### 💊 **【藥物緊急處理】**

```
**自殺風險評估（Emergency）：**
├── Benzodiazepines → 急性焦慮/激動控制
├── Haldol (Haloperidol) → 躁動鎮定（肌鬆劑輔助防肌肉痙攣）
└── Clozapine → 治療抵抗型精神分裂症（必須定期測白血球）

⚠️ 重要：自殺風險評估需包含「計畫、手段、意願」三要素！
📌 記憶法則：«Plan + Means + Will = Suicide Risk!»
```

#### ⚠️ **臨床警示**

**【高風險情境】**
1. **自殺過渡期（高危窗口）**
   - 🧠 **重點記憶**：
     * 急性期以被害妄想為主（不自殺）
     - 病情穩定後反而較易自殺（希望升起→面對現實痛苦）
     - 藥物轉換期最危險（症狀控制但未完全康復）
   - ⏰ **觀察指標**：情緒突然「變好」可能是假性改善

2. **精神緊急狀況處理流程**
   - 📝 **標準作業程序**：
     * Step 1: 安全評估（移除危險物品）
     * Step 2: 穩定情緒（Benzos/Antipsychotics PRN）
     * Step 3: 環境隔離保護
     * Step 4: 團隊會診（精神科/社工/家屬）

#### ❌ **易錯點**

- **❌ 錯誤觀念**：「病人出院後自殺風險降低」
  - ✅ **正確認識**：約 50% 自殺發生在出院後 3 個月內！需追蹤支持系統

#### 📚 **Chapter CH3-4: 生命統計與衛生教育**

##### **🌟 核心考點**

```
【生命統計】
├── Crude Birth Rate = (總出生數/年中人口) × 1000
├── Mortality Rate = (死亡數/年中人口) × 1000
└── Prevalence Rate = 現有病例數/調查人數

💡 國考重點：Prevalence = Incidence × Duration → 慢性病盛行率高≠新發案件多!
```

##### **🎓 衛生教育原則**

```
【5A 模式】
├── Assess → 評估需求與理解程度
├── Advise → 提出健康建議
├── Agree → 達成共同目標
├── Assist → 協助執行
└── Arrange → 安排追蹤計畫

⚠️ 臨床警示：衛生教育失敗常見原因→「資訊過量」!
📌 記憶法則：Less Is More! (5-10 個重點即可)
```

└── MERS/CORS → 防護裝備 + 感染控制

##### **🌟 核心考點**

```

├── 微生物指標 → Coliform < 10 CFU/L

【食品安全五大重點】

├── 煮熟 (Cook) → 中心溫度>75°C
└── 冷卻 (Chill) → 2 小時內冷藏

💡 國考重點：食物中毒前十大病原體→諾羅病毒、沙門氏菌、金黃色葡萄球菌!
📌 記憶法則：「清潔、分開、煮熟、冷藏」= 4C of Food Safety!
```

### 💊 **【抗憂鬱藥物・內分泌調節精選題庫】**

#### Topic 1: TCAs（三環類抗憂鬱藥物）機制

> **題目 #35**: 下列何者不是三環類抗憂鬱藥物之作用機制？
> - A. **抑制正腎上腺素與血清素再回收** → ✅ *正確機制*
> - B. **抑制單胺氧化酶之作用** → ❌ *這是 MAOI（單胺氧化酶抑制劑）的機制，不是 TCAs！*
> - C. **阻斷毒蕈鹼性受體** → ✅ *副作用來源（M 受体拮抗）*
> - D. **阻斷組織胺受體** → ✅ *副作用來源（嗜睡）*

✅ **答案 B**: TCA 的三大作用：
1. 抑制 NET (norepinephrine transporter)
2. 抑制 SERT (serotonin transporter)
3. Antagonist at M receptors, H1 receptors, α1 receptors

---

#### 💊 **抗憂鬱藥物分類與副作用比較表**

| 藥物類別 | 代表藥物 | 主要機制 | 特徵副作用 |
|---------|---------|---------|-----------|
| **TCAs** | Amitriptyline、Nortriptyline | NET + SERT inhibition | Anticholinergic, cardiac toxicity |
| **SSRIs** | Fluoxetine、Sertraline | SERT inhibition only | Sexual dysfunction, activation |
| **SNRIs** | Venlafaxine、Duloxetine | NET + SERT (dose-dependent) | Hypertension, nausea |
| **MAOIs** | Phenelzine、Tranylcypromine | MAO-A/B inhibition | Tyramine reaction, dietary restrictions |

#### 🧬 **【內分泌調節與精神疾病關聯】**

> **題目 #7**: 副甲狀腺素（PTH）作用在腎小管時會產生下列何種反應？
> - A. **刺激遠曲小管對鈣離子的再吸收** → ✅ *正確機制*
> - B. 刺激近曲小管對磷酸鹽的再吸收 → ❌ *PTH 抑制 PO₄²⁻重吸收*

✅ **精神疾病內分泌關聯**:
- **甲狀腺機能低下（Hypothyroidism）** → 抑鬱症狀加重
- **Cushing 症候群**（高皮質醇）→ 躁狂、焦慮症狀
- **糖尿病控制不良** → 神經病變影響認知功能

#### 🧬 **【內分泌疾病與精神行為】**

## 🧠 **【Phase 9.5】精神疾病・社區衛生 - 國考題目對接詳情（113 年第 1 次）**

### **🔬 Part I: 神經傳導物質與精神疾病**

#### **Q1: 帕金森氏症（Parkinson's disease）- 基礎醫學 1101-Q4**
> **題目：** 帕金森氏症（Parkinson's disease）的顫抖症狀肇因為何？
> 
> A. 腦內缺乏多巴胺（dopamine deficiency）✅
> B. 腦內多巴胺過高（dopamine excess）
> C. 腦內缺乏血清素（serotonin deficiency）
> D. 腦內血清素過高（serotonin excess）
> 
> **✅ 正確答案：A**
> 
> **📖 解析：**
> - 帕金森氏症的核心病變是黑質緻密部多巴胺神經元退化
> - 伏隔核（striatum）的多巴胺不足導致基底節迴路失衡
> - 症狀：顫抖、僵直、運動遲緩、姿勢不穩（TRAP）
> 
> **🎯 GVW 策略應考重點：**
> - 藥物作用機制：左旋多巴、多巴胺受體激动劑、MAO-B 抑制劑
> - 副作用：盤腦症候群（dyskinesia）、精神症狀
> 
> **🧠 記憶法則：**「Parkinson = Dopamine deficiency」

#### **Q2: 搖頭丸（Ecstasy/MDMA）- 基礎醫學 1101-Q33**
> **題目：** 下列對於毒品搖頭丸之敘述，何者最不適當？
> 
> A. 可抑制血清素之再回收 ✅
> B. 結構類似安非他命
> C. 可能引起高熱症
> D. 英文簡稱 NMDA ❌（正確應為 MDMA）
> 
> **✅ 正確答案：D**
> 
> **📖 解析：**
> - MDMA 機制：增加血清素、多巴胺、去甲腎上腺素的再釋放 + 抑制再回收
> - 高風險症狀：高熱症（hyperthermia）、脫水、齙齒磨損
> - NMDA 是另一類受體（如 ketamine 的作用標的）
> 
> **🎯 GVW 策略應考重點：**
> - 藥物濫用相關題目常考：MDMA→血清素耗盡→憂鬱後遺症

### **💊 Part II: 藥物代謝動力學（Psychopharmacology）**

#### **Q3: Fexofenadine vs. Diphenhydramine - 基礎醫學 1101-Q35**
> **題目：** Fexofenadine 較不具鎮靜安眠作用，原因為何？
> 
> A. 較難通過血腦障壁 ✅
> B. 對 H1 受體的選擇性增加
> C. 抑制中樞 GABA 受體
> D. 體內半衰期短
> 
> **✅ 正確答案：A**
> 
> **📖 解析：**
> - 第一世代抗組織胺（如 Diphenhydramine）→ 脂溶性高，易穿越血腦障壁 → 鎮靜作用
> - 第二世代抗組織胺（如 Fexofenadine、Loratadine）→ 极性高，難穿越血腦障壁 → 較不鎮靜

### **🏙️ Part III: 社區衛生重要概念**

#### **Q4: 社區性肺炎（Community-Acquired Pneumonia, CAP）**
> **題目：** 引起典型社區性肺炎最常見的致病菌為何？
> 
> A. 金黃色葡萄球菌
> B. 肺炎鏈球菌 ✅
> C. 流行感冒嗜血桿菌
> D. 黴漿菌
> 
> **✅ 正確答案：B**
> 
> **📖 解析：**
> - CAP 常見病原體：肺炎鏈球菌（最常見）、黴漿菌、流感嗜血桿菌
> - 醫院內肺炎（HAP）：金黃色葡萄球菌較常見
