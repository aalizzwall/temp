🌟 一、精神疾病護理 (High Yield)

1.1 精神分裂症 (Schizophrenia)

#### 🌟 核心考點 【診斷要點】 - Flesch Reading Ease: 7.2 (中等難度) - Key Symptoms: - 阳性症状：幻聽、幻覺、妄想、思维混亂 - 阴性症状：情感平淡、社交退縮、意志缺乏 - 病程: 至少持續 6 個月才算慢性化 【藥物治療】💊 ``` 第一代抗精神病藥 (典型)： ├── Haldol (Haloperidol) — 首選急性期躁動 ├── Thorazine (Chlorpromazine) — 嗜睡副作用大 └── Trilafon (Fluphenazine) — 半衰期長 第二代抗精神病藥 (非典型)： ├── Risperdal (Risperidone) — 首選，代謝副作用小 ├── Seroquel (Quetiapine) — 嗜睡、低血壓風險 ├── Zyprexa (Olanzapine) — 體重增加、糖尿病風險高 └── Geodon (Ziprasidone) — QTc 延長風險，需監測 💡 高頻考點： - Risperdal 是唯一代謝副作用較小的非典型藥物 - Clozapine 用於治療抵抗型精神分裂症（必須定期測白血球） ``` 【副作用記憶法則】 🧠 ``` 「5D+1H」： ├── Dry (乾) — 流體分泌減少 ├── Dizzy (暈) — 低血壓 ├── Dysphagia (吞嚥困難) — 自主神經障礙 ├── Depressed Mood (情緒低落) — 內臟性幻聽後的情緒問題 ├── Diabetes (糖尿病) — 體重增加/代謝綜合症 └── Hyperprolactinemia (高泌乳激素) — 溢乳/陽萎 📝 臨床警示：老年人使用第一代藥物需特別注意「Dizziness + Dry」，容易跌倒！ ``` #### ⚠️ 臨床警示 【高風險情境】 1. 自殺風險評估 - 🧠 重點記憶：精神分裂症病人病情穩定後反而較易自殺（急性期以被害妄想為主） - ⏰ 觀察指標：治療中藥效過渡期最危險 2. 藥源性肺炎風險 - 🩺 長期服用抗精神病藥者，需定期檢查肺部功能 - 💊 Olanzapine、Clozapine 較高風險 3. 代謝綜合症監測 ``` Zyprexa + Seroquel -> 高風險群體 └── 需追蹤：體重/BMI/血糖/血脂/HOMA-IR ``` 4. 藥物相互作用 - 🔄 CYP450 系統代謝影響： - Olanzapine (CYP1A2, 2D6) -> 與 SSRIs 合併使用需小心 - Quetiapine (3A4) -> 與 Statins 有相互作用 #### ❌ 易錯點 - ❌ 錯誤觀念：「精神分裂症病人需要隔離治療」 - ✅ 正確認識：社會功能恢復才是治療目標，應鼓勵社區融合 - ❌ 錯誤觀念：「所有抗精神病藥都會導致體重增加」 - ✅ 正確認識：Risperidone、Ziprasidone 的風險較低 ---

1.3 憂鬱症 (Major Depressive Disorder)

#### 🌟 核心考點 【診斷要點】 - DSM-5 Criteria: 至少 5 項症狀，包括： - 情緒低落或興趣丧失（2 項必須） - 睡眠改變 - 食慾改變 - 疲勞/精力不足 - 自我貶低/罪惡感 - 思考能力下降 - 自傷/自殺意念 【藥物選擇策略】💊 ``` SSRIs (首選): ├── Sertraline (Zoloft) — 首選，適合合併焦慮 ├── Escitalopram (Lexapro) — 副作用最小，老年人首選 └── Citalopram (Celexa) — QTc 延長風險 SNRIs: ├── Venlafaxine (Effexor) — 高劑量才有 SNRI 效果 └── Duloxetine (Cymbalta) — 適合合併慢性疼痛 Bupropion (Wellbutrin): └── 不影響性功能，適合合併吸煙癮斷戒需求 📝 臨床警示：抗抑鬱藥物需「啟動期 (2-4 週)」才見效，避免病人中途中斷！ ``` #### ⚠️ 臨床警示 【高風險情境】 1. 自殺風險評估 - 🧠 重點記憶：治療初期（第 2-4 週）情緒改善前，自殺意念可能反而增強！ - 📋 監測指標：每週一次自殺風險評估表 (SISSI) 2. 藥物停用症候群 ``` Paroxetine + Sertraline -> 高風險（需慢慢減量） Fluoxetine + Bupropion -> 較低風險（長半衰期） ``` 3. 血清素症候群警訊 🚨 ``` Triad Symptoms: ├── 自主神經興奮：心悸、發汗、震顫 ├── 精神狀態改變：焦慮、意識混亂 └── 肌肉超活化：肌攣、反射亢進 💡 處理：停藥 + 支援性護理，嚴重者需 ICU 觀察 ``` #### ❌ 易錯點 - ❌ 錯誤觀念：「抗抑鬱藥物沒有成癮問題」 - ✅ 正確認識：雖無苯二氮芪類的依賴性，但停藥仍需慢慢減量 - ❌ 錯誤觀念：「所有憂鬱症都需要抗鬱劑治療」 - ✅ 正確認識：輕度憂鬱可用心理治療 + 生活型態調整 ---

🌟 二、社區衛生護理 (High Yield)

2.1 傳染病社區照護 🦠

#### 🌟 核心考點 【高優先級傳染病】 ``` HIV/AIDS: ├── PEP (暴露後預防): 72 小時內開始，服用 4 週 ├── PrEP (暴露前預防): 每月檢查 HIV RNA + 腎功能 └── ART (抗逆轉錄病毒治療): 三種藥物組合 📝 臨床警示：HIV+ 孕婦可透過 C-section 降低垂直傳染率至 <1%! Tuberculosis (TB): ├── Initial Phase: RIPE (Rifampin, Isoniazid, Pyrazinamide, Ethambutol) ├── Continuation Phase: RH (Rifampin, Isoniazid) └── 監測指標：肝功能 + 視神經（Ethambutol） 📝 臨床警示：異物性結膜炎需隔離治療，避免傳染他人！ ``` #### ⚠️ 臨床警示 【高風險情境】 1. 抗逆轉錄病毒藥物副作用 ``` Integrase Inhibitors (首選): ├── Dolutegravir — 最常見，副作用小 └── Raltegravir — 懷孕安全 Protease Inhibitors: └── 高膽固醇/胰島素抵抗風險 💡 監測：每 3 個月一次完整血液檢查 + 腎功能 ``` #### ❌ 易錯點 - ❌ 錯誤觀念：「HIV 陽性者無法正常工作」 - ✅ 正確認識：現代 ART 治療下，病毒載量檢測不到=不傳染他人（U=U） ---

2.2 慢性病管理 🏙️

#### 🌟 核心考點 【糖尿病社區照護】 ``` Medication Hierarchy: ├── Metformin (首選) — GI副作用 + B12缺乏風險 ├── SGLT2 Inhibitors (新選擇) — 腎病/心臟病保護 ├── GLP-1 Agonists — 體重減少效果好 └── Insulin — 終末期需使用 💡 臨床警示：SGLT2 抑制劑有酮酸中毒風險（即使血糖正常）! 【高血壓社區照護】 ├── First Line: ACEI / ARB / CCB / Thiazide ├── Second Line: Combinaion Therapy (雙重治療) └── Goal BP: <130/80 mmHg (高齡者可放宽至<140/90) 📝 臨床警示：ACEI + ARB 組合不可同時使用（腎衰竭風險）! ``` #### ⚠️ 臨床警示 【高風險情境】 1. 藥效相互作用監測 - 🔄 Metformin + IV造影劑 -> 急性肾衰竭風險 - 💊 Insulin + 感染/壓力 -> 需調整劑量 2. 藥物依從性評估 ``` 高流失率群體： ├── 年輕族群（認為自己不會生病） └── 多重共病者（服用>5 種藥物） 💡 解決方案：簡化製程 + 藥學教育 ``` #### ❌ 易錯點 - ❌ 錯誤觀念：「血糖正常就不用吃降糖藥」 - ✅ 正確認識：藥物是為了保護器官功能，不能隨意停藥 ---

🌟 三、精神疾病社區復健 🔄

3.1 復健原則

#### 🌟 核心考點 【復健階段劃分】 ``` 急性期 -> 過渡期 -> 社區融合期 -> 自我管理期 ↑ ↑ ↓ ↓ ┌─────┼─────┬────────┴──────────┐ │ 醫院治療 + │ 社會支持系統建立 │ 家屬心理教育 │ └─────┘ │ │ 【重點記憶】：社區復健的核心是「賦權（Empowerment）」! ``` #### ⚠️ 臨床警示 【高風險情境】 1. 藥物減量策略 ``` Step-down Strategy: ├── 保持原藥劑量的 50% -> 2-3 週 ├── 轉換為長效製程 -> 1-2 週 └── 維持最低有效劑量 -> 長期追蹤 💡 臨床警示：突然停藥可能引發「撤藥症候群」! ``` #### ❌ 易錯點 - ❌ 錯誤觀念：「精神疾病病人沒有工作能力」 - ✅ 正確認識：適度復健可恢復社會功能，需早期介入 ---

🌟 四、注意缺陷過動症 (ADHD) 🧠⏱️

#### 🌟 核心考點 【診斷要點】 - DSM-5 Criteria: 至少 6 個症狀（兒童）/ 5 個症狀（成人），持續 >6 個月 - 三大類型： - ✅ Combined Type (混合型) — 最常見，注意力不足 + 過動衝動 - ⚠️ Predominantly Inattentive Type (注意缺失型) — 曾被稱為「ADD」 - 🔥 Predominantly Hyperactive-Impulsive Type (過動衝動型) — 成人較少見 【藥物治療】💊 ``` 興奮劑類（首選）： ├── Methylphenidate (Ritalin, Concerta) — 短效/長效選擇多 │ └── 副作用：食慾不振、生長延遲、睡眠障礙 │ └── Amphetamine (Adderall, Vyvanse) — 合併使用較無效果時考慮 └── 副作用：焦慮加重、心悸 非興奮劑類（二线）： ├── Atomoxetine (Strattera) — 首選用於有物質濫用史的病患 │ └── 起效慢（4-8 週），需監測肝功能 │ └── Guanfacine/Clonidine — 用於合併焦慮/過動症狀 💡 高頻考點：Concerta 使用 Osmotic Release System (OROS) 技術，每日一次即可！ ``` #### 🧠 【記憶法則：ADHD 藥物副作用口訣】 ``` 「三個 A」： ├── Appetite Loss (食慾不振) — Methylphenidate 常見副作用 ├── Agitation (焦慮/激越) — 高劑量時易出現 └── Anxiety (焦慮) — 成人 ADHD 較常合併 📝 記憶法則：「三個 A」-> 都以 A 開頭！考試快速排除選項用！ ``` ---

🌟 五、創傷後壓力症候群 (PTSD) 💔

#### 🌟 核心考點 【診斷要點】 - DSM-5 Criteria：需經歷創傷事件 + 四階段症狀（>1 個月） ``` Stage 1: Re-experiencing (反覆經驗) ├── Flashback (閃回) — 無法控制的侵入性回憶 ├── Nightmares (惡夢) — 典型睡眠障礙 └── Avoidance (迴避) — 主動迴避與創傷相關刺激 Stage 2: Hyperarousal (過度警覺) ├── Hypervigilance (高度警戒) ├── Startle Response (驚跳反應) — 聽到爆炸聲嚇一跳 ├── Sleep Disturbance (睡眠障礙) └── Irritability (易怒/憤怒爆發) Stage 3: Negative Changes (負面認知改變) ├── Guilt/Shame (罪惡感/羞恥感) ├── Emotional Numbing (情感麻木) — 無法感受到喜悅 └── Blaming Self/Others (自責/歸咎他人) Stage 4: Affective Problems (情緒問題) └── Depressive Symptoms (憂鬱症狀) — 合併憂鬱需綜合治療 ``` 【藥物治療】💊 ``` 第一線選擇： ├── SSRIs: Sertraline, Paroxetine — FDA 批准用於 PTSD │ └── 記憶法則：「SER-PAR」-> Sertraline + Paroxetine │ └── SNRIs: Venlafaxine — 二线選擇 第二線選擇（合併症使用）： ├── Prazosin (褪黑激素受體拮抗劑) — 改善噩夢/閃回 │ └── 臨床警示：需監測低血壓 │ └── PRN: Benzodiazepines (短期使用) — ⚠️ 有爭議，不建議長期使用 └── 記憶法則：BDZ -> Benzo DiAzipine（短效藥） 💡 高頻考點：PTSD 病人需同時使用「藥物 + 心理治療」效果最佳！ ``` 【心理治療】 🧠 ``` 證據支持的心理治療： ├── CBT (認知行為治療) — 第一線選擇 ├── EMDR (眼動統合治療) — 針對創傷記憶再處理 │ └── 記憶法則：Eye Movement + Dissociation -> EMDR │ └── Prolonged Exposure Therapy (延遲暴露) — 逐步面對恐懼刺激 ``` ---

🌟 六、成癮問題處理 💉🍷

#### 🌟 核心考點 【酒精依賴】🍷 【戒斷症狀評估】 ``` CIWA-Ar Scale (70-99 分 = 中重度戒斷): ├── Tachycardia (心動過速) — HR >100 bpm ├── Hyperhidrosis (多汗) ├── Tremor (震顫/手抖) — 經典徵象！ ├── Visual/Hallucinatory Symptoms (幻視/幻覺) │ └── 💡 記憶法則：CIWA -> C-I-W-A-A-r -> "See I Want A Alcohol" │ └── Seizure Risk (癲癇風險) — 高劑量後 24-72 小時 💡 記憶法則：戒斷症狀出現時間 = 最後飲酒後 6-48 小時！ ``` 【藥物治療】💊 ``` 急性期：BENZODIAZEPINES (首選!) ├── Chlordiazepoxide (Librium) — 長效，適合門診使用 │ └── 記憶法則：「Chlor」-> 氯（穩定）+ 苯二氮 │ ├── Lorazepam (Ativan) — ⚠️ 肝臟代謝不良者首選 │ └── Glucuronidation（不需 CYP450） │ └── Diazepam (Valium) — ⚠️ 老年人避免（長半衰期） 維持期治療： ├── Naltrexone — 拮抗鴉片類受體，減少渴求感 │ └── 記憶法則：「Nal」-> "No Alcohol" │ ├── Acamprosate — 調節 GABA/Glutamate 平衡 │ └── 副作用少，適合長期維持 │ └── Disulfiram (Antabuse) — ⚠️ 避免酒精（雙硫醇反應！） └── 記憶法則：喝酒後會「吐」（Disulfiram -> Dis-Suffer） ``` 【鴉片類成癮】🚬💉 【藥物替代療法】 💊 ``` Methadone (美沙酮): ├── ✅ 優點：長效（24-36 小時），可口服，減少針灸行為 │ └── ⚠️ 缺點：需定期監測（呼吸抑制風險） │ Buprenorphine (丁丙諾菲): ├── ✅ Suboxone = Buprenorphine + Naloxone │ ├── 高親和力 -> 部分激動劑 │ └── Naloxone 皮下注射時誘發戒斷（防止濫用） │ 💡 記憶法ظة：「Sub-Boxone」-> "Under Box = Partial Effect" │ └── Lozenges (舌下錠) — 適合門診使用 💡 高頻考點：Suboxone 使用方式需指導病人！(Swallow -> 吞下，非注射/咀嚼) ``` 【尼古丁成癮】🚬 【戒煙療法】💊 ``` Nicotine Replacement Therapy (NRT): ├── Nicotine Patch (尼古丁貼片) — ⭐⭐⭐ (持續釋放) │ └── 💡 記憶法則：「Patch -> Place」-> 放置於皮膚上 │ ├── Gum/Lozenges (尼古丁咀嚼膠) — 短效，應用作戰！ │ └── 記憶法則：「Gum -> Go Now!」-> 立即緩解 │ ├── Inhaler/Nasal Spray — ⚠️ 較貴，適合重度依賴者 │ └── Combination Therapy (貼片 + 咀嚼膠) — ✅✅ 最推薦！ 💡 記憶法則：「Combi -> Combo」-> 組合拳效果最佳！ ``` 【藥物治療速記卡】🃏

🌟 七、老年性精神疾病 🦴🧠

7.1 老年失智症 (Dementia)

📘 Chapter 10: 情感障礙病人的護理 (Mood Disorders)

#### 🌟 核心考點
障礙類型典型症狀病程特徵治療首選
重鬱症 (MDD)情緒低落、興趣喪失 >2 週單極性：只有抑鬱SSRIs + CBT
躁鬱症 (BD)躁狂 + 抑鬱交替發作雙極性：兩極循環心境穩定劑優先
#### 💊 【藥物選擇策略】 ``` MDD（重鬱症）： ├── SSRIs: Escitalopram/Lexapro -> 首選（副作用最小） ├── SNRIs: Venlafaxine -> 合併慢性疼痛/焦慮時 └── Bupropion -> 性功能副作用大、需提神者 BD（躁鬱症）： ├── Lithium -> 預防自殺風險最高（腎/TSH 監測） ├── Valproate -> 快速循環型首選（孕婦禁忌！） ├── Lamotrigine -> 抗抑鬱效果好，但啟動要慢（SJS 風險） └── Carbamazepine -> 肝毒性需監測 💡 高頻考點： - Lithium -> TSH + Renal Function (腎+甲狀腺) - Valproate -> Teratogenic (孕婦禁用！) - Lamotrigine -> Slow Start! (SJS Risk) ``` #### ⚠️ 臨床警示 【高風險情境】 1. Lithium 中毒徵兆 - 📌 記憶法則：「LIthium + 5+ 症狀」-> LIght-headed, Incoordination, Tremor, Irritability, Nausea - ⚠️ 目標血濃度：0.6-1.2 mEq/L（急性躁狂可達 1.5） - 💡 國考重點：發燒/脫水會減少腎排泄 -> Lithium 中毒！ 2. Valproate 致畸性警告 - 🩺 所有育齡期女性患者：計劃怀孕前需調整用藥 - 📝 記憶法則：«No Baby in the Lab!» (Lab = Liver) -> 肝毒性高！ 3. 快速循環型 BD 治療失敗 - 🧠 重點記憶：年發作<4 次為「單相型」，>4 次需調整策略為「心境穩定劑加強版」 #### ❌ 易錯點 - ❌ 錯誤觀念：「躁鬱症只需要抗抑鬱藥」 - ✅ 正確認識：單一抗抑鬱藥可能誘發躁狂發作！必須搭配心境穩定劑 ---

📘 Chapter 13: 危機處理與意外事件的護理 (Crisis Intervention)

#### 🌟 核心考點
危機類型緊急措施長期介入監測重點
自殺風險評估1:1 陪伴、移除危險物品CBT + 藥物治療情緒、計畫、手段可得性

(除非必要才入住) ``` ##### ⚠️ 臨床警示 【高風險情境】 1. 長照機構感染控制失敗 - 🧠 重點記憶：輪椅、床板易積塵，需定期消毒 - 💊 護理措施：標準預防措施 + 隔離防護 2. 老人虐待徵兆識別 - 🩺 不明瘀傷、營養不良、恐懼眼神 - 📝 記憶法že：«Unexplained Injury = Abuse Alert!» --- #### 📚 Chapter CH11: 社區心理衛生 ##### 🌟 核心考點 ``` 【心理衛生三要素】 ├── 心理健康 -> 情緒穩定、認知正常 ├── 社會適應 -> 人際關係良好、工作/學習表現正常 └── 道德倫理 -> 行為符合社會規範 【社區復元服務核心價值】 ├── Hope -> 希望恢復 ├── Autonomy -> 自主決定 ├── Identity -> 身份認同（非「病人」標籤） └── Purpose -> 生活目標（工作/志願活動等） 💡 國考重點：社區復元≠醫療治愈！核心是「功能恢復」+「社會融入」! 📌 記憶法則：Recovery = Hope + Autonomy + Identity + Purpose! ``` --- #### 📚 Chapter CH12: 學校衛生與護理 ##### 🌟 核心考點 ``` 【學生健康監測】 ├── 視力檢查 -> 近視高盛行率（螢幕時代） ├── 聽力篩檢 -> 早期發現中耳炎/語言障礙 └── 牙齒健康 -> 蛀牙、牙周病預防 【校園心理健康服務】 ├── 霸凌防治 (Anti-bullying) ├── 自殺防制 (Gatekeeper Training) └── 危機介入流程 ⚠️ 臨床警示：學生壓力源前十大->學業、人際、家庭衝突! ``` --- #### 📚 Chapter CH13: 職業衛生與護理 ##### 🌟 核心考點 ``` 【職業相關疾病】 ├── 肌肉骨骼傷害 (MSDs) -> 重複動作、不良姿勢 ├── 職業倦怠 (Burnout) -> 工作壓大過勞 ├── 職業暴露風險 -> 化學品、生物危害物質 【職場心理健康促進】 ├── 壓力管理課程 ├── 員工協助方案 (EAP) └── 人機工程改善 💡 國考重點：職業安全衛生法 + 勞工保健相關法規需掌握! 📌 記憶法že：MSD -> Musculoskeletal Disorders = Repetitive Motion + Poor Ergonomics! ``` --- #### 📚 Chapter CH14: 傳染病防治與護理 ##### 🌟 核心考點 ``` 【傳染病控制三鐵桶】 ├── 發現病例 (Find Cases) -> 積極篩檢、通報 ├── 隔離治療 (Isolate & Treat) -> 切斷傳播鏈 └── 預防接種 (Vaccinate) -> 群體免疫 【重點感染症】 ├── HIV/AIDS -> 抗病毒藥物 + 減少傳染風險行為 ├── TB -> RIPE 治療 + 密接者追蹤 ├── Hepatitis B/C -> 疫苗（B）、干擾素/直接滅活劑（C） └── MERS/CORS -> 防護裝備 + 感染控制 ⚠️ 臨床警示：多重抗生素處方會導致「超級細菌」問題！ 📌 記憶法že：RIPE -> Rifampin, Isoniazid, Pyrazinamide, Ethambutol = TB Regimen! ``` --- #### 📚 Chapter CH15: 環境衛生與食品衛生 ##### 🌟 核心考點 ``` 【水質標準】 ├── pH 值 -> 6.5-8.5（世衛建議） ├── 微生物指標 -> Coliform < 10 CFU/L └── 化學物質 -> Lead, Mercury, Asbestos 限制 【食品安全五大重點】 ├── 清潔 (Clean) -> 洗手、環境消毒 ├── 分離 (Separate) -> 生熟分開避免交叉污染 ├── 煮熟 (Cook) -> 中心溫度>75°C └── 冷卻 (Chill) -> 2 小時內冷藏 💡 國考重點：食物中毒前十大病原體->諾羅病毒、沙門氏菌、金黃色葡萄球菌! 📌 記憶法則：「清潔、分開、煮熟、冷藏」= 4C of Food Safety! ``` ---
