精神與社區衛生護理學 📚

🌟 核心考點總覽 (涵蓋 110-115 年國考)

本主題重點：以下知識點在護理師國家考試中頻繁出現，需特別記憶與理解！

A 診斷標準 (DSM-V)
  • 持續 2 週以上情緒低落、對興趣減失（至少 5/9 症狀）
B 抗憂鬱劑選擇
  • SSRI 類藥（fluoxetine, sertraline）、SNRI（venlafaxine）
C 副作用管理重點
  • 初服期嗜睡、性功能障礙、體重增加需密切監測
A 陽性症狀特徵
  • 幻聽、妄想、思維破裂（與現實脫離）
B 陰性症狀辨識
  • 情感淡漠、意志缺失、社會功能退化
C 抗精神病藥選擇
  • 第一代（haloperidol）、第二代（risperidone, olanzapine）
A 潘朵斯量表評估
  • GAD-7 症狀嚴重度分級（0-20 分：
    0-4：無、5-9：輕微、10-14：中度、15-20：嚴重）
B SSRIs 適應症
  • 廣泛性焦慮障礙、社交焦慮、恐慌發作（SSRI 為首選）
C 心理治療方法
  • 認知行為療法 CBT、正念減壓 MBSR
A 躁期特徵判定
  • 情緒高揚（3-4 週以上）、話多、睡眠需求減少
B 鬱期特徵判定
  • 情緒低落、活動減慢、自我價值感降低（與憂鬱症類似）
C 心境穩定劑選擇
  • Lithium（躁鬱症）、Valproate（女性優先）、Lamotrigine
A 健康教育需求分析
  • 目標群體、現行問題、可用資源評估（SWOT 分析）
B 參與式方法應用
  • 焦點團體討論、社區工作坊、居民代表會議
C 成效評估指標
  • 知識增進率、行為改變率、健康狀況改善（PRECEDE-PROCEED 模式）
A 支持團體運作方式
  • 同儕支持、經驗分享、問題解決技巧（T-Step 訓練）
B 家訪服務內容
  • 精神狀況評估、用藥遵從性檢查、社會資源連結
C 社區緊急應變
  • 自殺危機介入（SAD PERSONS 量表）、暴力行為預防
A 失智症認知篩檢
  • MMSE（1-30 分，＜24 分異常）、MoCA（≤26 分異常）
B 行為精神症狀管理
  • 激越、游走、攻擊性行為需藥物與環境調整並重
C 照顧者支持服務
  • 照顧者喘息服務、情緒支持團體、經濟補助申請協助
A ADHD 診斷標準
  • 注意力缺陷（6+ 症狀）、過動衝動（6+ 症狀）、持續≥6 個月
B 用藥治療選擇
  • Methylphenidate（第一線）、Atomoxetine（替代用藥）
C 家長教育重點
  • 正向增強、時間表建立、環境刺激減少
A 職業倦怠評估工具
  • Maslach Burnout Inventory（MBI）、Job Content Questionnaire
B 壓力管理技巧
  • 時間管理矩陣、正念減壓、工作界限設定
C 組織支持系統建置
  • EAP（員工協助方案）、團體諮商、彈性工作安排
A 標準防護措施
  • 洗手七步驟、無菌手套穿戴、個人防護具（PPE）
B 隔離種類選擇
  • 接觸隔離、飛沫隔離、空气隔離（依照傳染病類型）
C 疫情追蹤標準
  • 接觸者追蹤、主動監測、篩檢策略制定（金三角模式）

🔍 考題回顧 (110-115 年精選)


【110 年】精神與社區衛生護理學
(題目：病人使用抗憂鬱劑（SSRI 類）後出現自殺傾向，最可能發生於？
(A)第 1-2 週 (B)第 3-4 週 (C)第 5-8 週 (D)3 個月後)
正解：(A) 第 1-2 週
解析：抗憂鬱劑初期可能加重焦慮與躁動，增加自殺風險。⚠️ 常考重點：用藥安全監控與副作用管理。
【111 年】精神與社區衛生護理學
(題目：精神分裂症病人出現幻聽，最適當之護理措施為？
(A)勸說忽略 (B)協助分辨聲音來源並給予情緒支持 (C)禁止使用抗精神病藥 (D)強行安撫)
正解：(B) 協助分辨聲音來源並給予情緒支持
解析：不應該否定病人的感受，而應協助其辨識並提供情緒支持。⚠️ 常考重點：精神症狀處理技巧與人際溝通策略。
【112 年】精神與社區衛生護理學
(題目：GAD-7 評量表計分＞14 分，屬於何種嚴重度？
(A)無憂慮 (B)輕微 (C)中度 (D)嚴重)
正解：(D) 嚴重
解析：GAD-7 分數＞14 分表示焦慮症狀嚴重，需醫療介入治療。⚠️ 常考重點：心理評估工具應用與臨床解讀標準。
【113 年】精神與社區衛生護理學
(題目：老年失智症病人出現游走行為，最可能原因為？
(A)尋求刺激 (B)環境適應困難/記憶衰退 (C)故意反抗 (D)睡眠過剩)
正解：(B) 環境適應困難/記憶衰退
解析：失智症患者常因空間認知障礙與記憶喪失而有游走行為。⚠️ 常考重點：老年精神問題辨識與處理策略。
【114 年】精神與社區衛生護理學
(題目：社區支持團體的運作功能不包括？
(A)同儕經驗分享 (B)問題解決技巧訓練 (C)藥物治療指導 (D)情緒表達空間)
正解：(C) 藥物治療指導
解析：支援團體側重於心理社會功能重建，不屬於專業醫療職責範圍。⚠️ 常考重點：社區健康促進與精神衛生活動運作架構。
【115 年】精神與社區衛生護理學
(題目：職業倦怠的評估量表為？
(A)SDS (B)MBI (C)EPDS (D)PHQ-9)
正解：(B) MBI
解析：Maslach Burnout Inventory 為國際標準之職業倦怠評估工具。⚠️ 常考重點：心理健康篩檢工具應用與職場精神衛生資源配置。
💡 記憶法則：
A(症狀/需求)→B(評估工具)→C(介入策略)
⚠️ 常考：憂鬱症、精神分裂症、社區健康促進三大主題
