內外科護理 📚

🌟 核心考點總覽 (涵蓋 110-115 年國考)

本主題重點：以下知識點在護理師國家考試中頻繁出現，需特別記憶與理解！

A 病發機制
  • 有效循環血量銳減導致組織灌注不足
B 代償反應
  • 腎素分泌增加→血管收縮、醛固酮分泌→水鈉瀦留
C 臨床表現
  • 脈搏細速 (＞100 次/分)、血壓下降、意識改變、尿量減少（＜25ml/h）
A 發炎反應啟動
  • 病原體入侵觸發免疫系統反應，釋放細胞因子（IL-1, IL-6, TNF-a）
B 全身徵象
  • 體溫異常（高熱或低溫）、呼吸急促（＞20 次/分）、心率加快（＞90 次/分）、白血球增多（＞12,000/mm³）
C SOFA 評分加劇
  • 器官功能受損（凝血異常、乳酸累積、意識混亂、呼吸衰竭）
A 病理解剖
  • 冠狀動脈阻塞導致心肌缺氧壞死，通常發生於心尖部
B ECG 變化
  • ST 段抬高、T 波倒置、Q 波出現（病理性 Q 波代表心肌壞死）
C 急救重點
  • ASPIRIN 立即嚼服、硝酸甘油舌下含化、β阻斷劑控制心率
A 病理機制
  • 肺泡-毛細血管膜通透性增加，大量蛋白性液體滲入肺間質與肺泡
B 臨床特徵
  • 雙側陰影（X 光）、呼吸困難、PaO₂/FiO₂＜300 mmHg、氧合指數下降
C PEEP 治療
  • 高濃度 PEER 可維持肺泡張力、改善通氣/血流比、防止肺塌陷
A 目的
  • 補充血容量、維持電解質平衡、改善組織灌注
B 監測指標
  • 中心靜脈壓（CVP）、尿量、乳酸濃度、意識狀態、皮膚溫度與顏色
C 臨床警訊
  • 心搏過速、血壓下降、肺部濕囉音、頸靜脈膨滿（提示肺水腫）
A 低鈉血症
  • 血清鈉＜135mEq/L，輕度（130-134）、中度（125-129）、重度（＜125）
B 成因
  • 水分過量、腎上腺功能不足、利尿劑使用、酒精戒斷等
C 處理重點
  • 補充生理食鹽水或濃縮鈉氯、嚴重病例需注射 Desamino DDAVP
A 穩定性心絞痛
  • 活動或情緒壓力時發作、休息或硝酸甘油緩解、持續時間＜15 分鐘
B 不穩定性心絞痛
  • 靜息時也發作、頻率增加、程度加重、持續時間延長（＞20 分鐘）
C ACS 鑑別
  • 肌鈣蛋白原陽性（1-4h 內）、CK-MB 升高（3-8h 開始）、ECG ST 段改變
A 早期診斷
  • qSOFA 評分（意識混亂、呼吸＞22 次/分、SBP＜100mmHg）≥2 分需懷疑
B 抗生素時機
  • 一經診斷應在 1 小時內給予適當抗生素、血培養前抽樣（避免污染）
C 血管活性藥物
  • Norepinephrine 首選、若無反應可合併 Vasopressin、去甲腎上腺素降低心率效果較佳
A 病因機制
  • 胰島素缺乏導致葡萄糖利用受阻、脂肪分解增加→酮體累積（酸中毒）
B 臨床表現
  • 多飲多尿、脫水徵象（口乾、皮膚彈性差）、呼吸急促深快（Kussmaul 呼吸）、意識改變
C 治療原則
  • 靜脈補充等張鹽水→5% 葡萄糖、胰島素靜脈 drip、鉀離子補充（避免高血鉀）
A 缺血性中風
  • 腦動脈阻塞導致局部缺氧壞死，黃金時間 4.5 小時內可進行溶栓治療
B 出血性中風
  • 腦出血、蛛網膜下腔出血，需要控制血壓、降低顱內壓
C tPA 適應症
  • ＜4.5 小時內發病、無禁忌症（近期手術、主動脈瘤、出血性中風史等）
A 評估工具
  • NRS 數值評量表（0-10 分）、VAPS 視覺類比量表、BPI 疼痛指數
B 藥物選擇
  • 對照藥（對乙酰氨基酚）、NSAIDs（布洛芬）、阿片類止痛藥（嗎啡）
C 多模式止痛
  • 結合局部麻醉、神經阻斷術、非藥物療法（冷敷、放鬆訓練）
A 清創處理
  • 移除異物、切除壞死組織、止血、使用抗菌敷料（銀離子、碘）
B 縫合技術
  • 皮層縫合（深層對合）、皮下縫合（減壓張力）、表皮貼片或膠帶
C 感染預防
  • 維持清潔乾燥、定期換藥、觀察紅腫熱痛、抗生素適時使用

🔍 考題回顧 (110-115 年精選)


【110 年】內外科護理學
(題目：下列何者為敗血症休克之早期徵象？
(A) 血壓穩定 (B) 脈搏細速且尿量減少 (C) 體溫正常 (D) 意識清醒)
正解：(B) 脈搏細速且尿量減少
解析：敗血症休克早期為代償期，腎素分泌增加導致血管收縮、醛固酮分泌增加水鈉瀦留，因此尿量減少；脈搏細速則是有效循環血量不足之表現。
⚠️ 常考重點：SIRS 判讀標準（體溫異常、呼吸＞20 次/分、心率＞90 次/分、白血球異常）
【111 年】內外科護理學
(題目：急性心肌梗死病人接受藥物治療，下列何者正確？
(A) 硝酸甘油舌下含化 (B) ASPIRIN 嚼服 (C) β阻斷劑控制心率 (D) ACEI 降低心室負荷)
正解：(A, B, C, D)
解析：急性心肌梗死急救重點包括 ASPIRIN 立即嚼服（抑制血小板凝聚）、硝酸甘油舌下含化（擴張冠狀動脈）、β阻斷劑控制心率、ACEI 降低心室負荷。
⚠️ 常考重點：ACS 鑑別診斷與溶栓治療適應症
【112 年】內外科護理學
(題目：ARDS（急性呼吸窘迫綜合症）之病理生理機制為何？
(A) 肺泡-毛細血管膜通透性增加 (B) 肺泡氣體交換減少 (C) 雙側陰影 (D) PaO₂/FiO₂＜300 mmHg)
正解：(A, B, C, D)
解析：ARDS 病理生理機制為肺泡-毛細血管膜通透性增加，大量蛋白性液體滲入肺間質與肺泡，導致氣體交換減少；雙側陰影（X 光）與 PaO₂/FiO₂＜300 mmHg 為診斷標準。
⚠️ 常考重點：PEEP 治療與氧合指數評估
【113 年】內外科護理學
(題目：輸液治療之主要監測指標為何？
(A) CVP (B) 尿量 (C) 乳酸濃度 (D) 意識狀態)
正解：(A, B, C, D)
解析：輸液治療監測指標包括中心靜脈壓（CVP）、尿量、乳酸濃度、意識狀態、皮膚溫度與顏色；肺水腫警訊為頸靜脈膨滿、肺部濕囉音。
⚠️ 常考重點：心功能評估與組織灌注指標
【114 年】內外科護理學
(題目：低鈉血症之處理原則為何？
(A) 補充生理食鹽水 (B) 濃縮鈉氯 (C) Desamino DDAVP 注射 (D) 口服補液)
正解：(A, B, C)
解析：低鈉血症處理重點為補充生理食鹽水或濃縮鈉氯，嚴重病例需注射 Desamino DDAVP；避免快速補充（防止腦水腫）。
⚠️ 常考重點：血鉀電解質平衡與腎上腺功能評估
【115 年】內外科護理學
(題目：糖尿病酮酸症候毒血症 (DKA) 之急救處理為何？
(A) 等張鹽水補充 (B) 胰島素靜脈 drip (C) 鉀離子補充 (D) 血液透析)
正解：(A, B, C)
解析：DKA 急救處理包括靜脈補充等張鹽水、胰島素靜脈 drip（促進葡萄糖利用）、鉀離子補充（避免高血鉀）；嚴重病例需考慮血液透析。
⚠️ 常考重點：Kussmaul 呼吸識別與脫水評估
💡 記憶法則：
A(病發機制)→B(代償/病理改變)→C(臨床表現/處理)
⚠️ 常考重點：
• SIRS 判讀標準（體溫異常、呼吸＞20 次/分、心率＞90 次/分、白血球異常）
• ARDS 病理機制（肺泡-毛細血管膜通透性增加、雙側陰影、PaO₂/FiO₂＜300 mmHg）
• ACS 鑑別診斷與溶栓治療適應症
• 輸液治療監測指標（CVP、尿量、乳酸濃度、意識狀態）
• 低鈉血症處理原則（補充生理食鹽水、濃縮鈉氯、Desamino DDAVP）
📌 考試重點：
內外科護理是護理師國考必考科目，建議每日複習並搭配題目練習！
⏰ 重點記憶：休克代償反應、敗血症 SIRS 判讀、ARDS PEEP 治療、輸液監測指標
📖 推薦複習順序：休克→敗血症→ARDS→急性心肌梗死→輸液治療→電解質異常
