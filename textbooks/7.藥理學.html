藥理學 📚

🌟 核心考點總覽 (涵蓋 110-115 年國考)

本主題重點：以下知識點在護理師國家考試中頻繁出現，需特別記憶與理解！

A ACEI 類藥作用
  • 抑制血管緊張素轉換酵素、減少醛固酮分泌（ACE-I：Captopril, Enalapril）
B Calcium Channel Blocker
  • 維脈擴張劑（Dihydropyridine: nifedipine）、心肌負荷減少（non-dihydropyridine）
C Diuretics 利尿劑分類
  • Thiazide（HCTZ, chlorthalidone）、Loop（furosemide、托拉塞米）、保钾（spironolactone）
A Sulfonylurea 降糖機制
  • 刺激胰島素分泌、禁忌：妊娠/1型糖尿病（glibenclamide）
B Biguanides 類藥
  • Metformin：抑制肝糖生成、提高 Insulin Sensitivity（二甲雙胍為一線治療）
C DPP-4 Inhibitors
  • Incretin 類似物、延緩 Glucagon-like peptide-1 分解（sitagliptin）
A Warfarin 監測指標
  • INR 目標值：2-3（VTE）、2.5-3.5（機械心臟瓣膜）
B NOACs 藥物分類
  • Dabigatran（Direct Thrombin Inhibitor）、Rivaroxaban/Apixaban（直接 Xa 抑制劑）
C Reversal Agent 選擇
  • Warfarin：Vitamin K, Prothrombin Complex Concentrate；NOAC：Idarucizumab（Dabigatran）
A Atypical Antipsychotics
  • Second Generation：Risperidone, Olanzapine, Quetiapine（較少 EPS 副作用）
B Typical Antipsychotics
  • First Generation: Haloperidol, Chlorpromazine（EPS 風險高：錐體外症候群）
C Tardive Dyskinesia 警訊
  • 不自主口面部動作、延遲治療（不可逆），建議定期監測 TD 量表評分
A Beta-lactams 類抗生素
  • Penicillins (Amoxicillin)、Cephalosporins（1-4 代抗菌譜擴展）
B Macrolides 特點
  • Erythromycin, Azithromycin（支氣管炎首選、妊娠可用）
C Aminoglycosides 注意事項
  • Gentamicin：耳毒性/腎毒性、需監測血中濃度與聽力變化
A SSRIs 作用機制
  • 選擇性五羥色氨酸再吸收抑制（Fluoxetine, Sertraline, Escitalopram）
B SNRIs 類藥特性
  • 雙重機制：NSE+SNRI (Venlafaxine, Duloxetine)
C MAOIs 注意事項
  • 單胺氧化酶抑制劑、食物限制（酪胺類）：Cheese Reaction（高血壓危機）
A NSAIDs 消炎止痛
  • Ibuprofen, Naproxen、Celecoxib（COX-2 選擇性抑制劑，副作用較少）
B Opioids 鎮痛特性
  • Morphine, Oxycodone：嗎啡受體激動劑、易成癮與呼吸抑制（需呼吸監測）
C Adjuvant Analgesics
  • 加減藥用：Gabapentin, Pregabalin（神經病理痛治療）
A Statins 作用原理
  • HMG-CoA reductase 抑制劑：Atorvastatin, Simvastatin、肝膽代謝酵素抑制（CYP3A4）
B Fibrate 類藥特性
  • Gemfibrozil：增加 LPL活性，降低TG（高血脂合併用時注意肌病風險）
C Ezetimibe 機制
  • 膽固醇吸收抑制劑、與 Statin 併用增強降膽固醇效果
A Sodium Channel Blockers
  • Carbamazepine, Lamotrigine：局麻素作用、抑制神經興奮性傳導
B GABA Agonists
  • Diazepam, Lorazepam, Gabapentin：增加氯離子通道開啟頻率（抗焦慮與肌肉弛緩）
C Valproic Acid 特性
  • 多靶點機制、廣譜適用、肝毒性/致畸性風險高（需監測肝功能與甲胎蛋白）
A TNF-alpha Inhibitors
  • Infliximab, Adalimumab、類風濕性關節炎（RA）、Crohn 病首選治療
B IL-12/23Inhibitors
  • Ustekinumab：結腸炎症症疾病（IBD）第二線治療選擇
C Rituximab 特性
  • B 細胞去蕪菁劑、自體免疫疾病（RA、紅斑性狼瘡、多發性硬化症）

🔍 考題回顧 (110-115 年精選)


【110 年】藥理學
(題目：使用 Warfarin 時需監測何種指數？
(A)INR (B)aPTT (C)血小板計數 (D)凝血時間)
正解：(A) INR
解析：Warfarin（香豆素類）抗凝血機制為抑制 Vit K 依賴性酵素、INR 為標準化監測指標。⚠️ 常考重點：抗凝血藥物監測與副作用處理。
【111 年】藥理學
(題目：下列何者為 SSRIs 類抗憂鬱劑首選適應症？
(A)躁鬱症 (B)重度憂鬱症 (C)精神分裂症 (D)帕金森症)
正解：(B) 重度憂鬱症
解析：SSRIs（Selective Serotonin Reuptake Inhibitors）為憂鬱症一線治療藥物，副作用較少。⚠️ 常考重點：抗憂鬱劑分類與臨床應用指徵。
【112 年】藥理學
(題目：使用 Statin 類降血脂藥物時需監測何種副作用？
(A)肝毒性 (B)腎功能減退 (C)耳毒性 (D)視神經炎)
正解：(A) 肝毒性
解析：Statins（HMG-CoA reductase 抑制劑）主要副作用為肝酵素升高，需定期監測肝功能。⚠️ 常考重點：降脂藥物安全監控與預防措施。
【113 年】藥理學
(題目：使用 MAOIs 類抗憂鬱劑時需注意何種飲食限制？
(A)乳製品 (B)香蕉 (C)酪胺含量高的食物 (D)水果)
正解：(C) 酪胺含量高的食物
解析：MAOIs 抑制單胺氧化酶，攝取酪胺高之食物會導致高血壓危機（Cheese Reaction）。⚠️ 常考重點：特殊藥物與飲食交互作用關係。
【114 年】藥理學
(題目：下列何種抗生素對腎毒性風險最高？
(A)Penicillin (B)Cephalosporin (C)Amphotericin B (D)Gentamicin)
正解：(D) Gentamicin
解析：Aminoglycosides 類抗生素主要副作用為腎毒性與耳毒性，需監測血中濃度。⚠️ 常考重點：抗生素分類特性與臨床注意事項。
【115 年】藥理學
(題目：使用 NSAIDs 止痛藥時需注意何種風險？
(A)胃黏膜損傷 (B)肝毒性 (C)耳毒性 (D)腎功能減退)
正解：(A) 胃黏膜損傷
解析：NSAIDs 抑制 COX 酵素導致胃保護機制喪失，易造成胃炎、消化性潰瘍與消化道出血。⚠️ 常考重點：止痛藥安全性評估與併發症預防。
💡 記憶法則：
A(作用機制)→B(適應症/禁忌)→C(副作用/監測)
⚠️ 常考：抗凝血、抗憂鬱、抗生素三大藥物分類應用
📌 考試重點：藥理學涵蓋臨床藥物機制、適應症與安全性評估三大主題，建議熟悉常用藥物與監測指標！
✨ 記憶提示：機制→用途→監測（M-U-M）三步驟學習架構
