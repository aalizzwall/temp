產兒科護理學 📚

🌟 核心考點總覽 (涵蓋 110-115 年國考)

本主題重點：以下知識點在護理師國家考試中頻繁出現，需特別記憶與理解！

A 膽紅素代謝機制
  • 肝臟未成熟、腸道菌相未建立導致結合性膽紅素減少
B 光療標準指徵
  • 總膽紅素＞15mg/dl、光療時保護眼睛與生殖器部位
C 核黃疸危險因子
  • 溶血性疾病（ABO/Rh 不兼容）、G6PD 缺乏症
A Apgar 評分系統
  • A(Appearance)、P(Pulse)、G(Grimace)、A(Activity)、R(respiration)，每分鐘評一次
B 呼吸刺激法
  • 拍打足底、擦抹背部（避免強光直射眼睛）
C 正壓人工呼吸設定
  • 潮氣量 6-8ml/kg、PEEP 5-7cmH₂O
A 母乳餵養好處
  • 免疫球蛋白、成長因子、降低感染風險（WHO 推薦：6 個月內純母乳）
B 哺乳姿勢選擇
  • 側臥式、坐姿、半躺式（減少頸部肌肉負擔）
C 奶水不足警訊
  • 尿布每天＜6 片濕潤、體重未增加（應增加 15-30g/day）
A BCG 疫苗注射部位
  • 左上臂三角肌外側（皮內注射）、24-48 小時形成紅色丘疹
B 脊髓灰質炎疫苗
  • 口服 OPV 或注射 IPV、6 個月完成三劑基礎劑量
C DPT 疫苗接種禁忌
  • 發燒＞38℃、急性感染期、神經系統疾病病史
A 產後大出血處理
  • 子宮按摩、縮宮素（Oxytocin）靜脈 drip、血庫備血
B 胎盤早離子處理
  • 手動清除、預防性抗生素治療、密切監測生命徵象
C 羊水栓塞警訊
  • 突然呼吸窘迫、凝血功能障礙（DIC）、低血壓休克
A 子宮收縮評估
  • 觸及下腹部硬如板子、無壓痛感（正常表現）
B 惡露觀察標準
  • 血性期（3-4 天）、漿液性期（7-14 天）、白色期（2-3 週）
C 產後併發症辨識
  • 子宮內膜炎、深靜脈血栓、產後憂鬱症篩檢（EPDS 量表）
A 乳房擠乳方法
  • C 字形拇指抓握、輕壓而非擠壓（避免乳暈疼痛）
B 吸奶器設定標準
  • 漏斗罩尺寸（14mm-25mm）、抽力適度調整
C 冷藏母乳保存期限
  • 冰箱冷藏（4℃）：3-5 天、冷凍庫（-18℃）：3-6 個月
A 頭圍測量標準部位
  • 眉弓至枕骨後部（橢圓規繞過最凸處）
B 正常新生兒體長
  • 平均 50cm、出生後每月增加約 2.5cm
C 體重下降警訊
  • 生理性減重＜10%、＞10%需醫療介入（脫水/感染）
A 唐氏症篩檢時間
  • 孕婦 15-20 週進行（雙勝期最佳）
B 羊水穿刺適應症
  • ＞35 歲、唐氏篩檢高風險、超音波異常發現
C 產前基因檢測種類
  • NIFTY/NIPT（無傷胎盤）、羊水穿刺、絨毛採樣
A 耳聲發射測試 (OAE)
  • 出生後 48 小時內完成、無痛快速檢查
B 腦幹聽覺誘發電位 (ABR)
  • OAE 未通過者需進行（評估神經傳導功能）
C 延遲篩檢處理
  • 3 個月內轉介語言治療、配戴助聽器或人工耳蝸

🔍 考題回顧 (110-115 年精選)
【110 年】產兒科護理學
(題目：新生兒出現黃疸，總膽紅素 18mg/dl 應採何種處理？
(A)口服葡萄糖水 (B)光療治療 (C)換血療法 (D)觀察即可)
正解：(B) 光療治療
解析：新生兒黃疸時，總膽紅素＞15mg/dl 應開始光療。⚠️ 常考重點：黃疸分類與處理標準（未結合性 vs 結合性）。
【111 年】產兒科護理學
(題目：新生兒 Apgar 評分第 5 分鐘得 8 分，屬於何種狀態？
(A)健康正常 (B)輕度呼吸困難 (C)中重度窒息 (D)需要立即搶救)
正解：(B) 輕度呼吸困難
解析：Apgar 評分第 5 分鐘仍低於 7 分顯示有呼吸循環問題，需密切監測。⚠️ 常考重點：Apgar 評分時間點與臨床意義。
【112 年】產兒科護理學
(題目：產後子宮觸摸發現質地柔軟且有壓痛感，最可能發生？
(A)正常恢復 (B)子宮收縮不良 (C)內陰道炎 (D)會陰撕裂)
正解：(B) 子宮收縮不良
解析：子宮應硬如板子、無壓痛感，柔軟且有壓痛為子宮收縮不良警訊。⚠️ 常考重點：產後大出血原因辨識與處理。
【113 年】產兒科護理學
(題目：母乳餵養母親乳汁過少之警訊不包括？
(A)尿布每天 6-8 片濕潤 (B)體重每日增加（15-30g） (C)新生兒吸吮次數＜8 次/day (D)哭鬧時乳房凹陷)
正解：(A) 尿布每天 6-8 片濕潤
解析：正常母乳餵養應尿布每天≥6 片濕潤，這是奶水足夠的指標。⚠️ 常考重點：哺乳評估標準與改善措施。
【114 年】產兒科護理學
(題目：新生兒聽力篩檢未通過之原因不包括？
(A)中耳積水 (B)外耳道堵塞 (C)聽神經發育不全 (D)乳頭皰疹)
正解：(D) 乳頭皰疹
解析：乳頭皰疹與聽力篩檢結果無直接關聯，應排除環境干擾因素。⚠️ 常考重點：聽力篩檢前準備與延遲處理標準。
【115 年】產兒科護理學
(題目：羊水穿刺適應症不包括？
(A)孕婦＞35 歲 (B)唐氏篩檢高風險 (C)超音波顯示神經管缺陷 (D)輕度貧血治療)
正解：(D) 輕度貧血治療
解析：羊水穿刺用於基因檢測與染體異常篩檢，非貧血治療之適應症。⚠️ 常考重點：產前診斷技術與倫理考量。
💡 記憶法則：
A(風險評估)→B(標準操作)→C(併發警訊)
⚠️ 常考：黃疸處理、Apgar 評分、新生兒聽力篩檢三大主題
📌 考試重點：產兒科護理學涵蓋新生兒照護與產後護理兩大領域，建議熟悉操作技術與緊急處理流程！
✨ 記憶提示：評估→監測→介入（E-M-I）循環監控
