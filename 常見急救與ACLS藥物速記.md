## 🚑 附錄：常見急救與 ACLS 藥物速記

急救藥物請務必用「臨床情境」與「致命心律」來記憶，這是急診與加護病房情境題的必考區。

### 1. 心血管急救與致命心律 (ACLS 核心)
| 藥物名稱 | 主要適應症 / 臨床情境 | 機轉與護理重點 |
| :--- | :--- | :--- |
| **Epinephrine (腎上腺素)** | **心跳停止 (無脈搏 VF/VT、Asystole、PEA)**<br>過敏性休克 | **急救霸主**。強效收縮血管、提升心跳與血壓。<br>*(註：過敏性休克首選給藥途徑為大腿前外側肌肉注射 IM)* |
| **Atropine (阿托品)** | **有症狀的心跳過緩 (Symptomatic Bradycardia)** | 阻斷副交感神經，**提升心跳**。<br>*(註：最大累積劑量為 3mg，對心臟移植患者無效)* |
| **Amiodarone (心律錠)** | **致命性心律不整 (VF / 無脈搏 VT)** | 鉀離子通道阻斷劑，**穩定心室心律**。<br>*(註：需使用專用避光且不含 PVC 的管路)* |
| **Adenosine (腺苷酸)** | **陣發性上心室頻脈 (PSVT)** | **心臟重開機**。會造成短暫的幾秒鐘心跳停止 (Asystole)，需事先衛教病人。<br>*(註：半衰期極短 <10秒，必須從最靠近心臟的靜脈快速推注，並緊接著推注 20cc 生理食鹽水)* |

### 2. 中樞神經急症與專屬解毒劑
| 藥物名稱 | 主要適應症 / 臨床情境 | 機轉與護理重點 |
| :--- | :--- | :--- |
| **Diazepam / Lorazepam (BZD類)** | **癲癇重積狀態 (Status epilepticus)** | 鎮靜、肌肉鬆弛，是**停止持續性癲癇發作的絕對首選**。 |
| **Naloxone (Narcan)** | **鴉片類/嗎啡中毒** | 競爭性拮抗鴉片受體。<br>*(註：作用時間比嗎啡短，需密切觀察是否再次陷入昏迷與呼吸抑制)* |
| **Flumazenil** | **BZD 類安眠藥中毒** | 競爭性拮抗 BZD 受體。 |

### 3. 代謝、電解質與嚴重過敏
| 藥物名稱 | 主要適應症 / 臨床情境 | 機轉與護理重點 |
| :--- | :--- | :--- |
| **50% Glucose (D50W)** | **低血糖昏迷** | 瞬間提升血糖。必須從粗大靜脈給予，以免引發靜脈炎。 |
| **Calcium Gluconate (葡萄糖酸鈣)** | **高血鉀症 (Hyperkalemia) 併發心電圖異常** | 本身不降鉀，但能**穩定心肌細胞膜**，爭取時間預防致命性心室顫動 (VF)。 |
| **Sodium Bicarbonate (NaHCO3)** | 嚴重代謝性酸中毒、高血鉀 | 鹼化血液。不可與含鈣藥物從同一條靜脈管路給予（會產生沉澱）。 |
| **Diphenhydramine** | 嚴重過敏反應 | 抗組織胺 (H1 阻斷劑)。常與類固醇 (Hydrocortisone) 併用作為急救第二線輔助。 |
